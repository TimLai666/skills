# 五大場景 Playbook

每個場景四段：準備重點、開場話術、常見僵局與解法、紅線清單。全部話術可直接改數字使用，台灣商務語境。

---

## (a) 薪資談判

### 準備重點

- **談 total compensation，不談單一月薪**：把底薪、保障月數、變動獎金、簽約金、股票／分紅、到職日、職級、遠端天數、教育訓練預算全部列成可交易變數。單價談不動的時候，包裡其他東西都能動。
- 行情三角定位：薪資調查報告（抓 P50 與 P75）＋同職級朋友／獵頭口頭情報＋該公司職級帶（面試過程側面問 HR「這個職位的敘薪帶大概是什麼範圍」）。開價錨在 P75 附近、有依據能講。
- BATNA 工程：至少讓一個其他機會進入流程。沒有第二個 offer 時，「現職」就是 BATNA——算清楚留下來的一年總值（含預期調薪），那是 reservation point 的底。

### 時序規則（比話術更重要）

1. **拿到口頭 offer 之後才談**。進入 offer 階段前，公司在評估你；offer 之後，公司已決定要你，槓桿反轉。面試中被問期望薪資的擋法見下。
2. **例外——讓對方先出價的判斷**：一般原則是資訊優勢在誰誰先錨。求職場景中公司對自家薪級帶的資訊永遠比你多，所以**預設讓公司先開**；只有當你手上有明確強 BATNA（現職高薪或另一 offer 數字）且高於該公司可能的帶寬中位時，才主動先錨。
3. **避免先報前一份薪水**：前薪會變成天花板錨。擋法：「我更希望以這個職位的職責範圍和市場行情來討論，而不是用我上一份的數字——那份工作的範圍和這個不太一樣。方便先了解這個職位的敘薪帶嗎？」若制度上必填，填區間並在備註欄寫「期望以職位行情議定」。

### 開場話術（拿到口頭 offer 後）

「很謝謝這個 offer，我對這個機會非常有興趣，也希望能盡快定下來。在總包的部分，依我查到的行情（講出處），加上這個職位要求的 X 和 Y，我原本期待的是【錨點數字】。我們可以整包一起看——底薪之外，簽約金、保障獎金、到職時間都有討論空間。」

要點：先表達意願（降低對方風險感）→ 給依據 → 給錨 → 開整包（給對方多個讓步出口）。

### 常見僵局與解法

- 「這已經是這個職級的上限」→ 校準型問題：「了解，那在職級框架內，哪些部分是有空間的？」轉向簽約金（一次性、不動薪級）、提前調薪審核（六個月後 review）、職級本身重議。
- 「你要先給我一個數字」（offer 前被逼報價）→ 給有出處的區間，錨在區間下緣仍高於目標：「依行情這個角色在 P 到 Q 之間，具體要看整包內容。」
- 拿到數字偏低又沒有第二個 offer → 不虛構競爭 offer（紅線），改用行情標準撐：「這個數字和（出處）的 P50 有落差，我想理解一下這是怎麼評的？」

### 紅線清單

- 不虛構其他 offer、不謊報現薪。
- 不在拿到書面 offer 前提離職。
- 口頭談定後 24-48 小時內要求書面確認，所有談到的項目（含口頭承諾的 review 時點）都要進 offer letter 或 email。
- 接受後不再回頭加碼（毀約式加碼會燒掉整個圈子的信用）。

## (b) 供應商採購議價

### 準備重點

- **多供應商槓桿是採購談判的 BATNA 主體**：談判前至少取得兩家正式報價（不是網頁價目表）。就算沒有換的打算，「已詢價」這件事本身就改變桌上力學——而且這是事實，不是虛張。
- **TCO 不只看單價**：把單價、運費、付款條件、最小訂購量、不良率成本、交期延誤損失、切換成本全部折算成年度總持有成本再比較。單價低 5% 但 MOQ 翻倍的報價可能更貴。
- 摸對方壓力：對方的產能利用率、季底業績、庫存水位。業務窗口的個人 KPI（通常是簽單金額或毛利率）決定他能讓什麼。

### 開場話術（對漲價通知）

「收到你們的調價通知了。在回覆之前，想先了解這次調整的成本結構依據——是哪個項目漲了多少？（要求依據）我們也同步在檢視年度採購配置（BATNA 訊號，據實陳述），希望找到一個雙方都能長期走下去的方式。」

### 長約換價的條件設計

用「確定性」交換「單價」，這是把餅做大的標準結構：

- 我方給：合約年限（1 年→2-3 年）、最低採購量承諾、預測資訊共享、提前付款。
- 對方給：單價折讓、價格凍漲條款、優先產能、彈性交期。
- 必配保護條款：**量價對賭要雙向**——沒達到承諾量的補價機制，對應交期未達標的罰則；**市場價格重審條款**（原物料指數波動超過 ±X% 時重議），避免長約在市場反轉時變成枷鎖。

### 常見僵局與解法

- 「原物料漲，大家都漲」→ 要求對應到公開指數：「原物料佔你們成本結構的比例是多少？指數漲了 12%，反映到售價應該是多少？」把漲幅從喊價變成算術。
- 對方咬死單價 → 轉多變數 MESO：同單價但付款 90 天、或同單價含運含保固、或降 3% 但簽兩年。
- 獨家供應商（無替代）→ 槓桿改放在「未來份額」與「新品導入權」：現在的量談不動，用下一個案子的入場資格談。同時啟動第二供應商認證，這是長期 BATNA 工程。

### 紅線清單

- 不拿假報價單壓價（違法風險＋一旦被戳破永久失去議價信用）。
- 品質與安全規格不進交換池。
- 砍到對方沒有合理毛利的價格不要簽——對方會從你看不見的地方（料件、工序）拿回來。
- 所有談定條件進合約或訂單條款，口頭折讓不算數。

## (c) B2B 合作談判

### 準備重點

- **價值主張先行**：合作談判的第一份文件不是條件表，是「這個合作為對方創造什麼」的一頁紙。對方內部推動者需要拿著它去說服他的老闆——你在替你的談判對手準備他的內部簡報。
- 拆解對方的決策鏈：窗口、使用單位、法務、財務、拍板者，各自在意什麼。合作案死在法務和財務的比死在桌上的多。
- 分潤／出資比例先建計算模型，讓每個百分比都對應到雙方投入（流量、人力、品牌、通路），談判時調參數而不是喊數字。

### 開場話術

「我們評估過，這個合作對你們最直接的價值是【對方視角的收益：新客群／營收線／成本下降，給估算】。我們這邊投入的是【我方投入】。今天想先對齊雙方的目標和在意的點，條件結構我準備了幾個方向，等方向對了再細談。」

### Pilot 條款設計

大合約談不下來時，縮小成 pilot 是標準破局解，但 pilot 條款要防「白嫖」：

- 明確範圍與期限（3-6 個月、限定區域或客群）。
- **預先寫好成功指標與轉正條件**：「達到 X 就進入正式約、條件為 Y」——否則 pilot 結束等於重新談判，前期投入變沉沒成本。
- Pilot 期間的 IP、客戶資料歸屬先講清楚。

### 退出條款

- 談判時最不想談、出事時最重要。必備：終止的觸發條件、通知期、既有客戶／訂單的處理、資料返還與刪除、競業與挖角限制的存續期。
- 談退出條款的話術（避免傷氣氛）：「這部分是保護雙方——把不合的成本先講清楚，合作起來才敢放心投入。」

### 常見僵局與解法

- 對方要獨家、我方不想給 → 有條件獨家：「獨家可以，綁最低業績承諾，未達標自動轉非獨家。」把獨家從身分變成要用業績租的權利。
- 分潤比例拉鋸 → 改變結構而非比例：階梯式分潤（量越大對方分越多）、或前期我方多拿後期反轉，對應雙方投入的時間曲線。
- 對方窗口支持但案子卡在上層 → 替窗口做武器：一頁 ROI 摘要、風險對照表、pilot 降風險方案。必要時提議雙方主管層對談一次。

### 紅線清單

- 沒有書面 MOU／合約前不投入不可回收資源。
- 不簽無限期、無條件的獨家。
- 客戶資料與核心 IP 的歸屬沒寫清楚不簽。
- 對方要求「先做再補約」超過一個月即停手。

## (d) 跨部門資源談判

### 準備重點

- **無金錢籌碼，交換物要先盤點**：跨部門談判的貨幣是——排期優先順序（我把你的需求往前排）、成果露出（報告與簡報中的掛名與能見度）、人力互援（借調、代訓）、資訊（我團隊的數據或客戶回饋先給你）、未來的支持票（你下季的提案我公開支持）。談判前列出我手上有哪些、對方缺哪些。
- 對方的語言是他的 KPI：先查對方部門這一季被考核什麼，把我的需求翻譯成對他 KPI 的貢獻。
- 共同上級的優先序是最終客觀標準：公司 OKR、年度重點專案清單，引用它比引用自己的急迫感有力。

### 開場話術

「我需要你們團隊在【時段】支援【具體資源】。我知道你們手上有【對方的在檔工作，顯示做過功課】。這件事對你們的價值是【翻譯成對方 KPI】，另外我可以【交換物：把你們 X 的需求排進我們下個 sprint／在月會上把這段成果掛你們團隊】。」

### 升級路徑設計（Escalation）

升級不是告狀，是把「兩個部門解不了的優先序問題」交還給有權排序的人。設計規則：

1. 升級前先給對方看升級的內容：「我打算把這個資源衝突整理給【共同主管】排優先序，這是我要送的摘要，你看事實有沒有寫錯。」——透明的升級不傷關係，偷襲式的才傷。
2. 升級文件只寫三塊：兩案的公司層級影響對比、資源缺口的具體數字、建議方案（含折衷版）。不寫對人的評價。
3. 升級是第三步不是第一步：先雙邊談、再拉入雙方主管同席、最後才送共同上級裁決。跳步會被貼標籤。

### 常見僵局與解法

- 「我們也很忙，排不進去」→ 縮小請求＋校準型問題：「如果整包排不進，先支援哪一塊對你們成本最低？」拿到部分承諾後再擴。
- 口頭答應、實際不動 → 把承諾公開化：會議紀錄寫明「X 團隊於 Y 日前提供 Z」，副本給雙方主管。公開的承諾才有執行力。
- 每次都是我求人 → 結構性解法：建立常態互援協議（每季互相預留 N 人日），把重複談判變成制度，談一次用一年。

### 紅線清單

- 不拿「跟老闆很熟」當籌碼壓人。
- 不承諾自己團隊做不到的交換物（跨部門信用破產比對外更難修）。
- 共識一律落在會議紀錄或 email，口頭約定視同不存在。

## (e) 租屋／商業租約

### 準備重點

- **市場行情舉證**：住宅抓內政部實價登錄（租賃）＋租屋平台同路段、同屋齡、同坪數 3-5 筆在租物件截圖；商用另加同商圈空置狀況（空店越多，房東 BATNA 越弱）。物件在市場上刊登超過一個月，議價空間顯著放大——刊登日期本身就是情報。
- 我方 BATNA：真的去看 2-3 間可簽的備選，拿到具體條件。「我看過 XX 路那間，條件是……」的具體程度決定可信度。
- 房東的利益不只租金：空置成本（每空一個月＝損失 8% 年租金）、租客品質（穩定、愛惜、不吵）、管理成本。長租、準時繳、自理小修繕，對房東都是可計價的價值——把自己包裝成「低風險租客」是議價籌碼。

### 開場話術

「我很喜歡這個物件，也希望長期穩定租下來。我查了這個路段的行情（出示實價登錄與在租比較），同條件大概在 X 到 Y 之間，這間開 Z 稍微高了一些。我的情況是（收入穩定／可簽兩年／可自理小修繕），如果租金能到【錨點】，我今天就可以付訂。」

### 附帶條件交換（租金談不動時的變數池）

| 我方給 | 換 |
|--------|-----|
| 簽約 2 年（降低房東空置風險） | 月租減 3-5%，或凍漲條款 |
| 一次付半年／年繳 | 月租折讓或免一個月 |
| 自理 5,000 元以下修繕 | 租金微降或設備更新（冷氣、熱水器） |
| 接受現況不要求重新粉刷 | 免首月或半月租金（裝修期免租） |
| 商用：接受較長租期＋提前解約金 | 免租裝潢期 1-3 個月、租金逐年階梯式 |

### 常見僵局與解法

- 「這個價很多人在問」→ 測真偽（deadline 戰術變形）：「了解，那我不耽誤你。我的條件是 X，今天有效，你如果其他人選定了再跟我說。」真有競爭者你本來就拿不到，假的話房東會回頭。
- 房東咬死租金 → 全面轉附帶條件：租金照開，但要求免租期、設備、含管理費、含車位。名目租金不變（房東對外面子保住），實質成本下降。
- 續約被漲租 → 用「換租客的成本」談：「重新招租的空置期加仲介費大概是 1.5-2 個月租金，我續兩年、維持原價，你實拿其實更多。」附上同路段目前在租行情。

### 紅線清單

- 看清楚再簽：漲租條款、提前解約違約金、修繕責任歸屬、押金返還條件、（商用）禁止轉租與使用用途限制。
- 口頭承諾（「之後幫你換冷氣」）一律寫進合約或 LINE／email 留痕。
- 住宅注意《租賃住宅市場發展及管理條例》對押金上限（二個月）與修繕責任的規定——違反法定底線的條款，談都不用談，直接要求修正。
- 付訂金前確認產權（謄本）與出租權限（二房東要求出示原約與轉租同意）。
