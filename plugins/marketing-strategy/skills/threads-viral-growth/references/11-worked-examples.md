# 11｜實戰範例集

> 把整個 skill 工作流程跑一遍，給 3 個完整範例。

## 範例 1：北市大行銷系 × 高中生 TA（課程任務）

### Step 1｜定位

```
主體：北市大行銷系
TA：高中生（17-18 歲），正在選系階段
人格：實戰、有點壞、社群感、不端著
目標：讓高中生記住北市大行銷系、引發留言、產生「想念這裡」的衝動
禁區：不貶低其他大學、不過度官腔
Slogan：「不只學行銷，而是學怎麼被世界看見」
```

### Step 2｜趨勢偵測（範例 2026/06/01）

```
今日趨勢備忘
─────────────────────────────────────
Threads 平台前 5 趨勢話題（假設）：
1. 學測放榜後填志願焦慮
2. AI 時代讀什麼系才有競爭力
3. 大學生暑期實習被當免費勞工
4. 文組出路議題（媒體系炎上）
5. 「我們系到底在學什麼」標籤梗

對應 TA 的圈層討論：
- 高中生在 Threads 焦慮選系
- 同學間互相比較申請的學校
- 對「文組沒未來」的言論很反感
- 想知道某個系「真實的樣子」（不是招生簡章）

可借用的句型：
- 「我們系不是你想的那樣」型貼文最近常見
- 「我以為 X，結果 Y」反差體
- 「老實說，[系所] 最累的不是 OOO」

可挑戰的觀點：
- 大家在說「行銷系很涼」 / 「行銷系沒用」
- 我們可以反向操作：行銷系真正在學的不是表面那些
```

### Step 3｜題材選擇

從上面選定題材：**「行銷系真正在學的，不是大家想的那樣」**

通過 4 道閘：
- ✅ TA 痛點：高中生想知道系所真實樣貌
- ✅ 品牌連結：直接帶出北市大行銷系
- ✅ 可討論：高中生會問、學生會 +1、其他系會反駁
- ✅ 不踩雷：沒有政治、性別、貶低問題

### Step 4｜情緒 + Hook

主導情緒：**共鳴**（讓大學生 +1，讓高中生「真的嗎」）
Hook 類型：**共鳴型 + 反差型**

寫 5 個 Hook 候選：

```
1. 很多人以為行銷系只是在做海報。
2. 行銷系最累的不是報告，是組員說：我都可以。
3. 我以為讀行銷系會學寫文案，結果學的是怎麼跟人說人話。
4. 高中生不知道的事：行銷系沒有「行銷」這門課。
5. 行銷系畢業才懂：我們不是學行銷，是學「為什麼有人會停下來」。
```

選定：**第 1 個**（最強 Hook、最有反差、最普及）

### Step 5｜貼文成型

用「共鳴型 + 反差型」結構：

```
─────────────────────────────────────
很多人以為行銷系只是在做海報。

大三的時候，
有一次去家裡的婚禮，
親戚問我：「行銷喔？那你會幫我設計名片嗎？」

我笑笑沒回。

但其實我們學的，
不是「會做東西」，
是「為什麼要做這個東西」。

學「在 0.5 秒內讓人停下來」，
學「怎麼讓對的人，看到對的訊息」，
學「為什麼有人願意分享你的故事」。

行銷不是設計，
是研究人為什麼會被打動。

你以為的行銷系是什麼？

#北市大行銷系
─────────────────────────────────────
```

字數：約 220 字 ✅
Hashtag：1 個 ✅
留言鉤：「你以為的行銷系是什麼？」✅

### Step 6｜時間 + 自留言

**建議時段**：週四 20:00

**第一則自留言**：
```
其實這篇是被問到第 N 次後寫的。
你最常被問到什麼會想翻白眼的問題？
```

**預想留言**：
1. 「我們系也常被誤會 +1」（同學共鳴）
2. 「我以為行銷就是打廣告」（高中生詢問）
3. 「廣告系才是做廣告好嗎」（其他系反應）
4. 「我家人到現在以為我會用 PS」（自嘲）
5. 「那真正在學什麼？」（求補充）

**T+15 分鐘劇本**：
- T+1 發自留言
- T+3 通知 2 個同學去留言（不是讚，是有觀點的留言）
- T+5-10 回所有早期留言
- T+10 到「大學申請」相關大帳號近期貼文下留「觸」的留言

### Step 7｜配套圖卡

**圖卡主標**（大字置中）：
```
很多人以為
行銷系
只是在做海報。
```

**副標**（小字）：
```
真正的行銷，
是研究人為什麼停下來。
```

**底部簽名**：
```
北市大行銷系 · #不只學行銷
```

## 範例 2：獨立咖啡店 × 上班族 TA

### Step 1｜定位

```
主體：[假設] 一間台北東區的獨立咖啡店「停午後」
TA：25-35 歲上班族，重視生活儀式感
人格：慵懶、文青、有點哲學感
目標：讓 TA 把這間店記住、想拍照打卡、想推薦給朋友
禁區：不太銷售業配味、不批評連鎖咖啡
```

### Step 2｜趨勢偵測（假設）

```
TA 圈層討論：
- 「下班後不知道該去哪」
- 「一個人吃飯」議題
- 連鎖咖啡愈來愈像辦公室延伸
- 找一個「真正放空的地方」很難
```

### Step 3-5｜題材 + Hook + 貼文

主題：**「一個人在咖啡店」**
情緒：**溫暖 + 共鳴**
結構：**故事型**

```
─────────────────────────────────────
昨天有個女生來，
點了一杯黑咖啡，沒拿筆電。

她坐在角落，
看著窗外，看了快兩個小時。
不滑手機，不看書。
就只是 看。

走的時候她跟我說：
「謝謝你們不問我是不是在等人。」

那一刻我才懂，
我們做的不是咖啡，
是一個「不用解釋自己在幹嘛」的地方。

每個 25 歲以上的人，
都需要一個可以單獨待著、
而且 不需要假裝在做什麼 的地方。

你有這樣的地方嗎？

#停午後
─────────────────────────────────────
```

### Step 6-7｜時間 + 自留言

**時段**：週日 21:00（上班族「明天又要上班」焦慮中找慰藉）

**自留言**：
```
我們最近開始推「沒目的座位區」，
是一個不能擺筆電、不能講電話的區。
只是讓人單純坐著。
有沒有人想試？
```

## 範例 3：個人創作者 × 25-30 歲女性 TA

### Step 1｜定位

```
主體：個人創作者（行銷顧問 / 自由工作者）
TA：25-30 歲，在組織裡升不上去或正在猶豫離職的女性
人格：直接、誠實、有實戰經驗、不雞湯
目標：建立「她懂我」的信任，引導報名訂閱電子報
禁區：不批評女性主義、不踩政治、不過度推銷
```

### Step 2-5｜題材 + Hook + 貼文

主題：**「升不上去不是你的問題」**
情緒：**共鳴 + 微爭議**
結構：**爭議型**

```
─────────────────────────────────────
我跟你說一個你可能不愛聽的：

很多 28-32 歲的職場女性，
卡關不是因為「能力不夠」。

是因為太能扛了。

公司會把「能扛事」的人留在原位，
把「會發聲」的人推上去。

你能扛 = 你被需要在那個位置
你能說 = 你被需要在更上面的位置

這不是公平不公平的問題，
是組織政治的事實。

如果你也卡了 2-3 年，
試試看下個會議「敢說話」，
而不只是「敢做事」。

被罵就被罵，
不被罵你會永遠停在那裡。

你也卡 2 年以上了嗎？
─────────────────────────────────────
```

### Step 6-7｜時間 + 自留言

**時段**：週日 21:00（上班族焦慮期）

**自留言**：
```
我每週都會寫一封「職場誠實信」給訂閱者，
聊這種「沒人在公開說但很真實」的事。
有興趣留言「+1」我私訊你。
```

## 範例的共同邏輯

對比三個範例，找出共同點：

1. **每篇 Hook 都有「鉤點」**：反差、爭議、共鳴感
2. **情緒鎖定一個主導**：不混合
3. **都用「我們/你」第一/第二人稱**：拉近距離
4. **都有開放式留言鉤**：邀請對話
5. **都連結到品牌主張**：但不業配
6. **字數都控制在 250 字內**：易讀

這就是這個 skill 的核心輸出標準。

---

## 給使用者的練習建議

每次寫貼文前，可以先回頭看這 3 個範例：

1. **拿主題 vs 範例對照**：你的主題跟哪個範例的結構最像？
2. **拿 Hook vs 範例對照**：你的 Hook 強度跟範例比起來如何？
3. **拿留言鉤對照**：你的結尾有沒有真的邀請對話？

寫多了會內化，到時候不用範例也能下筆。

---

## 一句話結論

> 範例不是抄襲的對象，是**校準的座標**。
> 你的貼文必須長得像你（品牌語氣），
> 但結構與情緒設計，跟範例同等級。
