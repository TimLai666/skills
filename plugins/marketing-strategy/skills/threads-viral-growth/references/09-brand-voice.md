# 09｜品牌語氣設計

> 寫 1 篇貼文要會抓 Hook。寫 100 篇貼文需要的是品牌語氣系統。

## 為什麼語氣決定長期經營成敗

讀者追蹤一個帳號，不只是因為某篇貼文有趣，而是因為**這個帳號的語氣讓他覺得有連結**。

```
單篇爆文 → 觸及大量陌生人
品牌語氣 → 把陌生人轉換成追蹤者
```

沒有語氣的帳號，就算偶爾爆紅也漲不到粉，因為讀者記不住「你是誰」。

---

## 品牌語氣的 5 個維度

### 1. 距離感（Formal ↔ Casual）

```
正式：「我們認為行銷的本質在於...」
親密：「我們最近想了一件事...」
口語：「老實說啦，行銷其實就是...」
廢話：「欸欸欸你聽我說，這件事真的...」
```

Threads 的甜蜜點：**親密 ↔ 口語之間**。太正式像政府網站，太廢話像 Dcard。

### 2. 銳利度（Soft ↔ Sharp）

```
柔和：「也許每個人的選擇都有他的道理。」
中立：「這件事其實沒有標準答案。」
銳利：「不選擇本身就是一種選擇。」
攻擊：「不選擇的人，活該被生活選擇。」
```

Threads 容易爆的：**中立 ↔ 銳利之間**。

### 3. 自信度（Humble ↔ Confident）

```
謙遜：「我可能不是最懂的人，但我覺得...」
平和：「分享一下我的觀察。」
自信：「我跟你說，這件事就是這樣。」
高傲：「不要再說行銷很簡單了，沒讀過的閉嘴。」
```

Threads 偏好：**平和 ↔ 自信之間**。

### 4. 情緒外顯度（Reserved ↔ Expressive）

```
壓抑：「這件事我不太想評論。」
平淡：「我覺得這件事還可以再討論。」
外顯：「我超不爽這件事！」
誇張：「我整個人都不好了！這什麼鬼！」
```

Threads 接受外顯，但要**真實**，不要表演。

### 5. 觀點立場度（Balanced ↔ Decisive）

```
平衡：「雙方都有道理，要看情境。」
偏向：「我比較傾向 A，但不否定 B。」
明確：「我選 A，這是我的立場。」
偏執：「只有蠢人才選 B。」
```

Threads 偏好：**明確立場 + 留討論空間**。

## 4 種常見品牌人格範本

### 人格 1：誠實學長姊型

**適合**：教育、知識分享、職場帳號、行銷系、人資、創業者

**語氣定錨**：
- 距離感：親密 + 口語
- 銳利度：中立 + 偶爾銳利
- 自信度：平和 + 自信
- 情緒外顯度：平淡 + 偶爾外顯
- 立場度：明確立場

**範例口吻**：
```
「老實說，行銷系最該學的不是創意，
是把抽象變具體的能力。

我大三才搞懂這件事。
你呢，搞懂了沒？」
```

**用詞偏好**：
- 老實說、其實、我自己也、欸、我跟你說
- 不用：學術語言、官方公告語氣

---

### 人格 2：街頭吐槽型

**適合**：年輕品牌、休閒餐飲、運動、潮流、街頭文化

**語氣定錨**：
- 距離感：口語 + 廢話
- 銳利度：銳利
- 自信度：自信 + 偶爾高傲（角色設定）
- 情緒外顯度：外顯
- 立場度：明確

**範例口吻**：
```
「跟你說一個秘密：
慣老闆永遠不會跟你開誠布公。

他們只會說『我們公司是大家庭』
然後叫你假日上班。」
```

**用詞偏好**：
- 跟你說、欸幹、真的、笑死、誰啊
- 不避諱：黑色幽默、自嘲、白眼

⚠️ 但不能變成「酸民人格」，要有觀點。

---

### 人格 3：溫暖陪伴型

**適合**：療癒、家庭、寵物、心理健康、生活風格、咖啡店

**語氣定錨**：
- 距離感：親密
- 銳利度：柔和 + 中立
- 自信度：謙遜 + 平和
- 情緒外顯度：外顯（但溫暖型）
- 立場度：平衡 + 偏向

**範例口吻**：
```
「凌晨兩點，我媽傳訊息：
『冰箱有湯，記得熱。』

她不會說想我，
但她記得我愛喝湯。

每個說『不要擔心』的人，
都在用自己的方式擔心。」
```

**用詞偏好**：
- 也許、其實、我想、那種感覺
- 多用比喻、留白、慢節奏

---

### 人格 4：專業權威型

**適合**：B2B 服務、顧問、投資、財經、科技

**語氣定錨**：
- 距離感：親密 + 口語（避免冰冷）
- 銳利度：中立 + 銳利
- 自信度：自信
- 情緒外顯度：平淡（克制）
- 立場度：明確（用數據撐）

**範例口吻**：
```
「2025 年很多人在問：AI 會不會取代行銷。

我的答案：
不會取代行銷，會取代『不會用 AI 的行銷人』。

實際數據：
頂尖代理商今年招的 junior，
有 73% 在面試時被問 prompt 設計能力。
這就是答案。」
```

**用詞偏好**：
- 我的答案、實際數據、根據觀察、結論
- 用數字、案例、引述，建立可信度

## 建立你的品牌語氣 SOP

### 步驟 1：寫品牌人格 3 個形容詞

選 3 個（不能超過 3 個）：

```
範例：
北市大行銷系 → 誠實的、有點壞的、社群感的
某咖啡品牌   → 溫暖的、慵懶的、文青的
某運動品牌   → 直接的、有挑戰性的、年輕的
```

### 步驟 2：寫品牌的「敵人」

明確說出**不是什麼**，比說「是什麼」更精準：

```
範例：
北市大行銷系 不是：高高在上的、官方公告式的、保守的
某咖啡品牌   不是：商業的、冷漠的、過度精緻的
```

### 步驟 3：定義常用詞 / 禁用詞

每個品牌都有自己的「術語」和「禁忌詞」。

**範例（北市大行銷系）**：
```
常用詞：
- 「老實說」「其實」「我們都」
- 「行銷不是 X，是 Y」（標誌句型）
- 「你有沒有發現...」
- 表情符號：適量用，但不要超過 2 個 / 篇

禁用詞 / 避用：
- 「敬請」「特此」「謹此」（官方語）
- 「乾貨」「劃重點」「滿滿的」（業配感）
- 過度的「！！！」「？？？」
- 「家人們」（中國風用語，台灣不吃）
```

### 步驟 4：寫一份「語氣對照表」

```
情境：要請大家留言
❌「請於下方留言給予回饋」
❌「歡迎大家留言」
✅「你也是嗎？留言給我看」
✅「我準備好被罵了，下面開戰吧」

情境：要分享觀點
❌「謹此分享個人淺見」
❌「以下分享我的看法」
✅「老實說，我覺得...」
✅「跟你說一個你可能不愛聽的」
```

### 步驟 5：建立「靈感倉」

把你看到的好句子、好 Hook、好留言收集起來。**這不是抄襲庫**，是讓你的語氣不斷對齊的素材。

工具：Notion / Apple Notes / NotebookLM。

## 一致性測試

每篇貼文發出前，跑這個檢查：

```
□ 這篇貼文的語氣，跟我上一篇貼文一致嗎？
□ 用詞風格符合我設定的「常用詞 / 禁用詞」嗎？
□ 如果讀者沒看到我的帳號名，能猜出是我嗎？
□ 跟我設定的 3 個品牌形容詞符合嗎？
```

如果第 3 題答案是「猜不出來」 = 還沒有真正的品牌語氣。

---

## 進階：用 ChatGPT / Claude 維持語氣一致

如果用 AI 輔助寫文，這樣設定 prompt：

```
你是 [品牌名] 的社群小編。

品牌人格：[3 個形容詞]
品牌不是：[3 個形容詞]
TA：[具體 TA 描述]
常用句型：[3-5 個]
禁用詞：[列清單]

請用這個語氣寫 Threads 貼文，主題：[X]
```

每次都用這個 prompt 模板（存起來），AI 產出的語氣會更一致。

## 常見錯誤

❌ **學別人語氣**：抄到一半變四不像。
✅ **找到自己的語氣**：哪怕粗糙，先有再說。

❌ **每篇換語氣**：「今天可愛、明天嚴肅、後天搞笑」 → 讀者搞不清你是誰。
✅ **80% 一致 + 20% 變化**：核心語氣穩定，偶爾延伸。

❌ **用官方語氣假裝親民**：「親愛的粉絲們」「我們團隊」這種詞會立刻拉開距離。
✅ **用第一人稱 / 對話口吻**：「我」「我們」「你」「老實說」。

❌ **怕失誤就什麼都不講**：結果發的都是無腦正能量。
✅ **明確立場 + 留討論空間**：你可以強硬，但要邀請對話。
