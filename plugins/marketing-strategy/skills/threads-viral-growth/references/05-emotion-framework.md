# 05｜情緒設計框架

> 沒有情緒的貼文 = 沒有靈魂的清單。Threads 是情緒平台，先選情緒，再選結構。

## 為什麼情緒決定擴散

Jonah Berger 在《Contagious》提出：**高情緒激活（high arousal）內容更容易被分享**。

關鍵不是「正向」或「負向」，而是「情緒喚起的強度」。

```
高喚起（容易擴散）       低喚起（不會擴散）
─────────────────       ─────────────────
憤怒、興奮、驚訝          滿足、平靜、放鬆
焦慮、共鳴、幽默          無聊、無感、中性
```

意思是：**寫一篇讓人心情平靜的貼文，不會爆**。寫一篇讓人想罵、想笑、想截圖傳給朋友的貼文，才會爆。

## 六大主導情緒（爆文最常用）

### 1. 共鳴（同感、被理解）

**擴散行為**：留言「+1」「就是我」「真的」+ 轉發給朋友

**最強場景**：TA 的日常痛點、生活中的小無奈、不被理解的時刻

**怎麼寫**：
- 寫出**具體場景**，越具體越打人
- 用「我們」「你」「大家」第二/第一人稱，不要「他們」
- 結尾留「你也是嗎？」的問句

**範例**：
> 「我們都在裝自己沒事，
> 然後在凌晨三點傳訊息給不該傳的人。
> 你最近一次傳是什麼時候？」

**勝率**：★★★★★（最穩的一種）

---

### 2. 憤怒（不爽、不公、雙標）

**擴散行為**：引用 + 長留言、罵留言、引發辯論

**最強場景**：社會亂象、職場雙標、被當盤子、體制問題

**怎麼寫**：
- 鎖定**明確的對象/現象**（不是空泛抱怨）
- 用堅定但不失邏輯的句式
- 不要寫到失控，要有「白眼」的味道，不是「咆哮」

**範例**：
> 「『我們是一個大家庭』
> 是慣老闆說過最不要臉的一句話。
> 家人不會叫你免費加班。」

**勝率**：★★★★（流量高但傷品牌風險也高）

**⚠️ 注意**：憤怒型容易失控。寫完再讀一次，把任何讓你後悔的詞砍掉。

---

### 3. 幽默（自嘲、白眼、產業內梗）

**擴散行為**：截圖傳給朋友、轉發、留言一起笑

**最強場景**：產業內笑話、自嘲、社會觀察、生活吐槽

**怎麼寫**：
- 幽默的核心是**反差**（預期 vs 現實）
- 自嘲 > 嘲諷他人（不傷人 = 更容易擴散）
- 不要解釋笑點，留白讓讀者自己領會

**範例**：
> 「老闆說『下班就下班』的時候，
> 你就知道今天要加班了。」

**勝率**：★★★★（適合所有品牌）

**⚠️ 注意**：幽默有文化差異。台灣讀者吃「白眼幽默」、「廢話文學」、「自嘲」；不吃「強行搞笑」、「玩諧音梗」。

---

### 4. 驚訝（反差、冷知識、沒想到）

**擴散行為**：留言「真假」「我不知道」「沒想到」+ 收藏 + 分享

**最強場景**：冷知識、反差體悟、業界內幕、數據驚奇

**怎麼寫**：
- 前半句設「合理預期」，後半句用「但其實…」翻盤
- 給一個具體**數字/事實**讓人記住
- 結尾留「你知道嗎？」/「你以為呢？」

**範例**：
> 「我以為長大會有答案，
> 27 歲才發現只有更多問題，
> 而且每個問題都比上一個更難。」

**勝率**：★★★（中等，但很安全）

---

### 5. 焦慮（落後、稀缺、不確定）

**擴散行為**：收藏（怕忘）、轉發給同處境的人、留言問細節

**最強場景**：競爭話題、限時、AI 時代焦慮、世代焦慮

**怎麼寫**：
- 點出**具體威脅**（不是泛泛的「現在很競爭」）
- 給一個**參考標準**（讓人對照自己有沒有落後）
- 結尾不要太絕望，要留一條出路

**範例**：
> 「你的同學在偷偷學 AI 提示工程，
> 而你還在猶豫要不要付 ChatGPT 的訂閱。
> 不是 AI 會取代你，是會用 AI 的同學會取代你。
> 從哪一個 prompt 開始學起？」

**勝率**：★★★（爆紅快，但消耗 TA 信任）

**⚠️ 注意**：焦慮型短期效果好，但用多了會讓品牌變「煽動者」。建議搭配建設性結尾。

---

### 6. 溫暖（被理解、被支持、被看見）

**擴散行為**：分享給朋友、留言感謝、追蹤帳號

**最強場景**：故事型貼文、節慶、紀念日、人物特寫

**怎麼寫**：
- 用**具體人物 + 場景**，不要泛泛說溫暖
- 不要硬煽情，**輕輕地**比重重地更打人
- 結尾留「你也曾經…嗎？」

**範例**：
> 「凌晨兩點，我媽傳訊息：
> 『冰箱有湯，記得熱。』
>
> 她不會說想你，但她會記得你愛喝湯。」

**勝率**：★★★（漲粉勝率最高）

## 情緒選擇決策樹

```
你的目標是什麼？
├── 想要最高觸及       → 共鳴 / 幽默
├── 想要爆量留言       → 爭議（憤怒衍生）/ 反問
├── 想要漲粉           → 溫暖（故事型）/ 共鳴
├── 想要強記憶         → 驚訝 / 反差
├── 想要轉換行動       → 焦慮（搭配解方）
└── 想要分享傳染       → 幽默 / 共鳴
```

---

## 一篇貼文 = 一個主導情緒

**禁止混合**。一篇貼文寫得又共鳴又憤怒又溫暖，結果是讀者抓不到主軸，誰都不想留言。

**做法**：
1. 先決定主導情緒（一個）
2. 全篇所有句子都服務這個情緒
3. Hook 用該情緒最強的句型
4. 結尾留言鉤呼應該情緒

**例外**：可以「主導情緒 + 微量輔助情緒」。例如以共鳴為主、結尾帶一點溫暖。但比例至少 80/20。

## 情緒強度檢核（寫完跑一次）

讀完整篇貼文，閉上眼睛問自己：

```
□ 我現在感覺到什麼情緒？（必須能講出來）
□ 這個情緒強度夠嗎？（夠 = 想留言/想截圖/想轉發其中之一）
□ 整篇情緒是一致的嗎？還是混亂？
□ 結尾有沒有把情緒收回 / 留討論空間？
```

如果讀完什麼情緒都沒有 = 整篇要重寫。

---

## 進階：情緒節奏

長一點的貼文（200+ 字）可以設計「情緒節奏」：

```
情緒節奏範例：
─────────────
第 1 段：吐槽 / 帶點憤怒        ← Hook，先抓住人
第 2 段：共鳴 / 場景鋪陳        ← 讓 TA 代入
第 3 段：轉折 / 帶一點驚訝      ← 製造記憶點
第 4 段：留言鉤 / 邀請對話      ← 收尾
```

不是每篇都要這樣設計，但長文一定要有起伏，不能整篇平。

## 安全區與危險區

| 情緒 | 安全度 | 注意事項 |
|---|---|---|
| 共鳴 | 🟢 高 | 大膽用 |
| 幽默 | 🟢 高 | 別開歧視玩笑 |
| 溫暖 | 🟢 高 | 別煽情過頭 |
| 驚訝 | 🟡 中 | 別誇大事實 |
| 焦慮 | 🟡 中 | 配建設性結尾 |
| 憤怒 | 🔴 低 | 用完一定要冷靜檢查 |

## 給品牌帳號的特別提醒

如果是品牌帳號（不是個人創作者），情緒選擇要更謹慎：

- **大量用**：共鳴、幽默、溫暖
- **適量用**：驚訝、焦慮
- **慎用**：憤怒（除非品牌人格就是「敢講」）

但**完全不用情緒也是錯的**。沒有情緒的品牌帳號 = 政府機關的廣告 = 沒人理。

關鍵是找到品牌**人格**對應的情緒組合。例如：
- 「正能量 / 教育」品牌 → 共鳴 + 溫暖 + 偶爾驚訝
- 「叛逆 / 街頭」品牌 → 幽默 + 憤怒 + 爭議
- 「專業 / 諮詢」品牌 → 驚訝（行業洞察）+ 焦慮（搭配解方）
- 「年輕 / 校園」品牌 → 共鳴 + 幽默 + 反差
