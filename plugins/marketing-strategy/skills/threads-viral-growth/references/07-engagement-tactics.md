# 07｜互動與留言策略

> 發文只是 30%，互動才是 70%。Threads 是對話平台，沒有對話就沒有流量。

## 核心觀念

```
單向廣播（FB/IG 邏輯）         ❌ Threads 不吃這套
雙向對話（Twitter/X 邏輯）     ✅ Threads 的 DNA
```

Threads 演算法把「能引發討論」當核心訊號。你發完文就閃人，演算法會把你當灌水帳號。

## 發文後關鍵 30 分鐘劇本

### T+0｜發文

確認：
- 內文沒外連（要連結就放第一則留言）
- 一個 hashtag 或不用
- 留言鉤明確
- 你接下來 30 分鐘有空在線

### T+1 分鐘｜自留言（必做）

立刻自己留第一則留言。這則留言的功能：

**選項 A：補充內容**
```
（例）對了，文中那個「組員說我都可以」的例子，
是我大三上組的悲劇真相。誰沒遇過？
```

**選項 B：放外部連結**
```
（例）想看完整版的可以來這 → [連結]
```

**選項 C：拋第二個討論點**
```
（例）其實還有一個更糟的情況：
組員突然失聯。你會怎麼處理？
```

**選 A 或 C**最常見。連結（B）只有真的有資源要分享才用。

**為什麼要自留言**：
- 製造「貼文已有留言」的視覺暗示，誘發路人也留言
- 自留言會被你的追蹤者看到，多一次曝光
- 演算法會把「自己回覆」也算成活躍訊號

### T+3 分鐘｜外援啟動

私下傳給 2-3 個朋友 / 同伴 / 互助群：
```
「我剛發了一則 Threads，
方便去留個有觀點的留言嗎？
不是按讚，是真的留一句話。」
```

**重點**：
- 真的觀點留言，不要「+1」「好讚」
- 至少寫 10 字以上
- 帶起對話的問題或補充

如果有經營「互讚群組」或「社群圈」，這就是用的時候。

### T+5–10 分鐘｜你自己回所有早期留言

**這一步最關鍵**。Buffer 研究指出，**回覆留言的帳號互動率提升 42%**。

回留言的原則：
- 不要只回「謝謝」「+1」
- 回得有內容、有觀點
- 可以反問：「你覺得呢？」「那你怎麼處理？」
- 連自己的留言也可以再回（接續討論串）

範例：

```
讀者留言：「我們組長還會說『大家加油』然後人就消失」

❌ 爛回覆：「太可怕了哈哈」

✅ 好回覆：「『大家加油』翻譯：你們去做，
           我去摸魚。
           遇過最高級的版本是『有問題隨時找我』
           然後就再也不上線。」
```

### T+10–15 分鐘｜海巡法

到其他熱門帳號 / 同類大帳號的近期貼文下，留**一則「觸」的留言**。

「觸」的意思：精闢、好笑、共鳴強、夠有觀點，會被該貼文的讀者按讚把它推到最前面。

**為什麼這樣做**：
- 那則熱門貼文的觀眾會看到你的留言
- 如果留言獲讚很多，會被推到留言區頂端
- 路人會點你的個人頁看「這人是誰」→ 追蹤轉換

**選擇貼文的標準**：
- 對方剛發 1-3 小時內（早期留言曝光更高）
- 對方主題跟你 TA 重疊
- 對方文末有開放式問題（你正好可以回應）

### T+30 分鐘｜檢查表現

打開貼文後台 / 數據看：
- 觸及多少？
- 留言量？
- 引用 / 轉發？

**如果起飛了**：考慮立刻發第二則延伸貼文，乘勢追擊。
**如果掛了**：不要立刻刪，等 6-12 小時看演算法有沒有後續推。

## 留言區經營（長期戰）

### 為什麼留言區 = 第二個內容區

每個留言都是貼文的延續。留言區夠精彩，會發生：
- 路人為了讀留言區停更久（停留時間 ↑）
- 路人想加入討論（留言量 ↑）
- 留言互相回覆（互動深度 ↑）

**演算法會把這些訊號都加分**。

### 留言區設計策略

**主動回覆每一則早期留言**（前 24 小時內的）。

**回覆順序**：
1. 先回**最熱的留言**（讓它的曝光繼續放大）
2. 再回**有觀點/有故事的留言**（引出更多人加入）
3. 最後處理**負面留言**（如果有）

**負面留言處理**：
- 不要刪除（會引起更大不滿）
- 不要對罵（傷品牌）
- 用幽默化解 / 直接認錯 / 拒絕參戰

範例：

```
負面留言：「廢文。」

❌ 刪除（用戶會發現，更不爽）
❌ 「謝謝你的指教」（被動侵略性，會更激怒）
✅ 「廢文的反義詞我也不知道是什麼欸，
    認真給我建議的話下次想看什麼？」
   （把對方拉回建設性對話）
✅ 直接已讀（如果留言沒理也沒讚，會自然沉下去）
```

### 引導留言的小技巧

**句尾不要結論式**：

```
❌ 「結論：行銷系就是要這樣讀。」
✅ 「你覺得呢？」
```

**留下「未完成」**：

```
❌ 「行銷系五大重點：① ② ③ ④ ⑤」
✅ 「行銷系四大重點：① ② ③ ④
    第五點我故意空著，你會放什麼？」
```

**直接點名 TA**：

```
❌ 「歡迎留言」
✅ 「正在準備學測的高三生，
    你們現在最焦慮的是什麼？」
```

## 海巡法（Comment Hacking）詳解

### 什麼是海巡法

不發新文，而是**到別人的熱門貼文下留高品質留言**，靠該貼文的流量帶自己曝光。

**證據**：行銷實測中，在 1 萬讚的爆文下留早期 + 觸的留言，自己那則留言可以衝到 2000+ 讚。對小帳號是最快的破圈方式。

### 海巡 SOP

1. **每天追蹤 5-10 個目標帳號**（同 TA、調性類似、互動高）
2. **設定提醒**：他們發文時你能在 1 小時內看到
3. **看到新貼文 → 寫一則「能接話」的留言**
4. **留言要點**：
   - 字數 20-60 字
   - 有自己的觀點 / 經驗 / 反差
   - 可以借勢，但不要硬抄
   - 不要打廣告
   - 不要 @ 自己（被當作蹭流量）

### 海巡的禁忌

❌ 留「同意」「+1」「好讚」這種低品質留言
❌ 在每篇貼文都留言（演算法會降權）
❌ 在留言裡塞外部連結 / 帳號
❌ 攻擊原 PO 的觀點（容易黑掉）
❌ 一直留一樣的話術（會被認出來）

### 海巡的進階：留言 = 創作

把每則留言當成迷你貼文寫。範例：

某美食帳號發了：「永和的豆漿真的全國 No.1 嗎？」

```
❌ 「+1 認同」
❌ 「我也覺得！」

✅ 「永和豆漿是台北人的鄉愁，
    不是真的最好喝，
    是凌晨 3 點還開、燒餅還是熱的。
    
    你問我永和豆漿好不好喝，
    我會說：看你幾點喝。」
```

這種留言會被讚到頂端，原 PO 也很有機會引用你。

## 串文（Thread）操作

一個觀點 350 字裝不下時，用串文。

### 串文 vs 單則貼文

| | 單則 | 串文 |
|---|---|---|
| 觸及 | 中 | 高（每則都有獨立觸及） |
| 適合 | 簡短觀點 | 複雜論述、故事、清單 |
| 留言 | 集中在主文 | 分散到每則 |
| 風險 | 低 | 中間斷掉讀者就走 |

### 串文設計原則

```
第 1 則：強 Hook，留懸念（必須能站得住獨立）
         ⬇
第 2 則：展開論述，給細節
         ⬇
第 3 則：深入 / 反差 / 轉折
         ⬇
最後一則：結論 + 留言鉤
```

**注意**：每一則都要能單獨被截圖傳閱（不能 100% 依賴前文）。

### 串文時機

- 第 1 則發完，**等 30 分鐘以上**再發第 2 則（讓第 1 則先有觸及）
- 或直接連發（適合連貫故事）

## 引用與轉發（讓別人擴散你）

### 怎麼讓人想引用你的貼文

引用 = 對方把你的貼文當成自己內容的一部分。為什麼會這樣做：

1. **你說出了他想說的話**（共鳴型）
2. **你說了他想反駁的話**（爭議型）
3. **你說了一個他覺得很重要的觀點**（驚訝型）

**設計重點**：寫貼文時想像「如果有人引用這則，他會加一句什麼？」如果你想得出 3 個版本，這篇就有引用潛力。

### 自己也要學會引用

看到爆文（不是自己的），可以引用 + 寫一句犀利觀點。

範例：

```
某帳號發了：「我覺得 30 歲還沒結婚沒什麼好擔心的。」

你引用 + 你寫：
「我同意。但更該擔心的是 30 歲還沒搞清楚
 自己要什麼。
 婚不婚是時機問題，
 自我是時間問題。」
```

這算「加值內容」，演算法給的權重很高，原 PO 看到也可能轉發你。

## 留言區危機處理

### 常見狀況

**狀況 1：留言被洗版負評**
- 先別反擊，看是不是內容真的有問題
- 如果是誤會，發補充說明（不要刪原文）
- 如果是攻擊，用幽默或冷處理

**狀況 2：被人引用 + 黑特**
- 點進去看對方什麼背景
- 如果是大帳號黑特你：要決定是回應還是冷處理
- 回應的話：堅定但有禮貌，不要被帶情緒

**狀況 3：留言突然爆量**
- 是好事！繼續回覆熱門留言
- 留意有沒有失控的言論（種族/政治/性別）需要管理
- 不要刪除善意的批評，會傷信任

## 數據追蹤：哪些指標看什麼

| 指標 | 看什麼 |
|---|---|
| 觸及 | 有沒有破圈 |
| 留言量 / 觸及 | 留言鉤設計得好不好 |
| 引用 + 轉發 | 內容能不能擴散 |
| 個人頁點擊 | Hook 有沒有強到讓人想看你是誰 |
| 新增追蹤 | 內容值不值得長期關注 |

**最該追求的指標**：**留言 / 觸及比 > 1%**

意思是：每 100 個看過貼文的人，至少有 1 個留言。低於這個 = 留言鉤太弱。

---

## 心法：把每則貼文當成「派對開場」

你發貼文不是「廣播」，是「在派對上開了一個話題」。

派對禮貌：
- 開了話題後**留下來和大家聊**，不是說完就閃
- 別人接話你要**認真接**，不是嗯哼帶過
- 如果聊得起勁，**繼續延伸**，不是強行收場
- 如果有人插話**很有趣，要 follow up**，不是冷掉

這就是 Threads 經營的全部哲學。
