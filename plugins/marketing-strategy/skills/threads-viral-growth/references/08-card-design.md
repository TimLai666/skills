# 08｜社群圖卡設計

> 在 Threads 上，能被截圖傳閱的圖卡，是貼文擴散的第二載體。

## 為什麼需要圖卡

純文字貼文很好用，但圖卡有獨特優勢：

1. **可截圖傳閱**：朋友會用 IG 限動分享 / 私訊截圖傳
2. **強記憶**：視覺比文字易記
3. **跨平台**：同一張卡可以丟 IG / FB / LinkedIn
4. **品牌印記**：用一致的視覺風格讓人記住「這是你」

## Threads 圖卡的視覺基因

### 5 大設計原則

```
① 黑白極簡    一張卡片只用兩個顏色（黑、白 + 一個 accent）
② 大字排版    主標放到極大，佔畫面 50%+
③ 一句話     一張卡 = 一個訊息（不要塞兩個觀點）
④ 易截圖     重要訊息集中在中間，不被 UI 切掉
⑤ 品牌簽名   底部有小字 logo / 帳號名 / 風格簽名
```

### 為什麼是「黑白極簡」

- **辨識度**：在彩色的動態牆上，黑白特別顯眼
- **權威感**：像 Apple Keynote、紐約時報，自帶高級感
- **易讀**：高對比讓人 0.3 秒讀完核心訊息
- **跨主題**：什麼題目都不衝突

例外：如果品牌已有強烈色彩識別（例如螢光綠、亮橘），可以用品牌色 + 黑/白。

## 圖卡尺寸與規格

| 用途 | 比例 | 像素 |
|---|---|---|
| Threads / IG 動態 | 1:1（正方） | 1080 × 1080 |
| Threads / IG 限動 | 9:16（豎屏） | 1080 × 1920 |
| 多圖輪播 | 4:5（建議） | 1080 × 1350 |

**最推薦**：4:5（1080 × 1350），佔螢幕版面最多，截圖效果最好。

## 三大圖卡版型

### 版型 A：超大字主視覺（最常用）

```
┌──────────────────────────┐
│                          │
│  [小字：北市大行銷系]    │
│                          │
│                          │
│  很多人以為              │
│  行銷系                  │
│  只是在做海報。          │
│                          │
│                          │
│  ─── （線）              │
│  真正的行銷，            │
│  是研究人為什麼停下來。  │
│                          │
│  [小 icon] [#hashtag]    │
└──────────────────────────┘
```

**特色**：
- 主標 = Hook 的關鍵字（最多 15 字）
- 副標 = 補一句觀點（最多 25 字）
- 頂部品牌名 + 底部簽名

### 版型 B：對話框 / 引用框

```
┌──────────────────────────┐
│  [品牌 logo]             │
│                          │
│  「                      │
│   最累的不是報告，       │
│   是組員說：             │
│   我都可以。             │
│                                    │
│                       」 │
│                          │
│  — 行銷系大三的真心話   │
│                          │
│  ─── （細線）            │
│  [小字 simulated UI:     │
│   ♥ ⌽ ↗  6,832]         │
└──────────────────────────┘
```

**特色**：
- 像引用名言一樣處理
- 可加「擬 Threads UI」元素（讚數、留言、轉發 icon）製造熟悉感
- 適合金句、共鳴型貼文

### 版型 C：清單型 / 數字型

```
┌──────────────────────────┐
│  讀行銷系 4 年            │
│  我想告訴高中生           │
│  這 5 件事                │
│                          │
│  ─                       │
│                          │
│  ① 不是會做海報           │
│  ② 不是文案最重要         │
│  ③ AI 不會搶你工作        │
│  ④ 敢上台是分水嶺         │
│  ⑤ 最該學會說人話         │
│                          │
│  ─                       │
│  [品牌 logo]              │
└──────────────────────────┘
```

**特色**：
- 適合多圖輪播：第 1 張是清單總覽，後面每張展開一個 point
- 用數字 / 編號維持節奏感

## 字體選擇

### 中文字體推薦

| 字體 | 用途 | 哪裡找 |
|---|---|---|
| 思源黑體 Bold / Heavy | 主標 | Google Fonts 免費 |
| 思源宋體 | 主標（古典感） | Google Fonts 免費 |
| 蘋方 / 微軟正黑 | 副標、內文 | 系統內建 |
| 標楷體 | 慎用，年輕族群會嫌「公務員」 | — |
| 王漢宗 / 文泉驛 | 注意版權 | — |
| 巴黎肥宋體 / 字嗨字體 | 設計感主標 | 部分免費 |

### 英文字體推薦

- **Inter / Roboto**：現代簡潔
- **Playfair Display**：高級感襯線
- **Bebas Neue**：街頭感大標

### 字級規則

```
主標：80–150 pt（佔畫面 50%+）
副標：32–48 pt
補充小字：18–24 pt
品牌簽名：14–20 pt
```

## Canva 操作建議

### 設計流程

1. 開新檔案 → 自訂尺寸 → 1080 × 1350
2. 背景：純黑 #000000 或純白 #FFFFFF
3. 加文字：主標思源黑體 Heavy 120pt，置中
4. 副標：思源黑體 Regular 36pt，主標下方
5. 頂部：品牌名小字 + 一條分隔線
6. 底部：簽名、icon、hashtag 小字
7. 匯出 PNG（壓縮品質 80%）

### Canva 常見地雷

❌ 用 Canva 預設模板（會被認出來「Canva 味」）
❌ 用太多 icon / 裝飾元素
❌ 用 3 種以上顏色
❌ 用 stock photo 當背景（沒個性）

## Affinity Designer / Photo（如果有用）

如果用 Affinity（廷榛在用 Affinity v3.0）：

```
1. 新檔 1080 × 1350 px
2. 背景純色填充
3. Text frame：主標 + 副標分層
4. 用 Character panel 調字距（Tracking 約 -20 ~ -50）
5. Export → PNG，色域 sRGB
```

**好處**：比 Canva 更精細，可以做出 InDesign 等級的排版。
**注意**：之前的對話顯示 Affinity 匯出有時跟其他軟體有相容性問題，建議匯出 PNG（不要用複合格式）。

## 多圖輪播設計

### 何時用多圖輪播

- 觀點需要分點（清單型）
- 故事有起承轉合
- 教學步驟

### 結構

```
圖 1：封面（最強 Hook）
圖 2：展開第 1 點
圖 3：展開第 2 點
...
圖 N-1：高潮點 / 反差
圖 N：CTA（留言/追蹤/分享）
```

### 設計重點

- 圖 1 必須能單獨站住（路人只看到第一張）
- 統一視覺風格（顏色、字體、版型一致）
- 圖底部加進度指示（1/5, 2/5...）讓人想滑下去
- 最後一張要有明確 CTA

## 動圖 / 短影音

如果想升級：

- **超短迷因影片**（3-5 秒）：靜態圖 + 簡單動效，TikTok 風格
- **打字機動效**：文字一個字一個字浮現（用 Canva 動畫 / CapCut）
- **TXT 風格**：白底黑字打字機動效（很多 Threads 大帳號在用）

工具：
- CapCut / 剪映：免費、好用
- Canva 動畫功能
- After Effects（專業）

## 圖卡與貼文的搭配策略

### 策略 1：圖卡 = 主文字
貼文內容 = 圖卡上的字。
適合：金句型、共鳴型，圖卡就是貼文。

### 策略 2：圖卡 = 視覺鉤
貼文寫長一點觀點，圖卡只放最 punch line 的一句話。
讀者被圖吸引 → 點開貼文讀詳情。

### 策略 3：圖卡 = 補充資訊
貼文是文字觀點，圖卡是數據 / 流程 / 清單。
適合：知識型、教學型。

## 圖卡品牌系統建議

長期經營，建議建立**圖卡品牌系統**（Brand System），保持視覺一致：

```
Brand System：
├── 主色：純黑 #000000 / 純白 #FFFFFF
├── Accent 色：[品牌色，例：#FF3E3E 螢光紅]
├── 主標字體：思源黑體 Heavy
├── 副標字體：思源黑體 Regular
├── 品牌簽名位置：右下 / 左下
├── 標準 hashtag：#[品牌名] #[主題標籤]
└── 版型：3-4 種固定版型輪用
```

一致的視覺會讓使用者**滑動態牆時 0.3 秒認出來是你**。長期累積等於品牌資產。
