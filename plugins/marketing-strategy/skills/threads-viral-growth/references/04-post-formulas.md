# 04｜爆文公式｜5 種結構模板

> Hook 抓住人，結構讓人停下來看完並留言。

## 五種爆文結構（依勝率排序）

| 結構 | 勝率 | 適合場景 | 互動類型 |
|---|---|---|---|
| 共鳴型 | ★★★★★ | TA 痛點、生活吐槽 | 留言 +1、轉發 |
| 爭議型 | ★★★★ | 反主流觀點 | 大量留言、引用辯論 |
| 故事型 | ★★★★ | 品牌建立、漲粉 | 共鳴留言、追蹤 |
| 清單型 | ★★★ | 知識/技巧整理 | 收藏、轉發 |
| 反問型 | ★★★ | 互動導向 | 留言參與 |

---

## 結構 1：共鳴型（爆文最常見）

**邏輯**：「TA 心裡早就想講，但沒人幫他講出來的那句話」。

### 結構

```
① Hook（共鳴/吐槽）           [前 1-2 行]
② 鋪陳情境（畫面感）          [2-3 行]
③ 核心觀點（一句話）          [1 行]
④ 留言鉤（問句或立場）        [1-2 行]
```

### 範例

```
最累的不是寫報告，
是組員說：「我都可以」。

明明是要一起想，
變成你一個人扛。
你提的他不回，他出的你不敢嫌。
最後熬夜寫完，他丟一句「辛苦了」。

行銷系的真本事，
不是會做簡報，是會逼出沉默組員的真心話。

你 那組 也是這樣嗎？
```

**為什麼會爆**：
- Hook 直接打中所有做過小組報告的人
- 情境鋪陳到讓人浮現臉孔
- 核心觀點把現象升級為「能力」，連結到品牌
- 結尾留言鉤是開放式「你那組也是」，誰看到都想留言

### 應用模板（直接套用）

```
[現象 Hook：最累/最氣/最孤獨的不是 X，是 Y]

[情境 1：具體畫面]
[情境 2：再加深一點]
[情境 3：到極致的那種感受]

[核心觀點：把這個現象拉到一個更普遍的層次]

[留言鉤：你也有過嗎？/ 你最受不了哪一種？]
```

## 結構 2：爭議型（留言最多）

**邏輯**：提出反主流觀點，引發兩派辯論。**留言量勝率第一**，但風險也高。

### 結構

```
① Hook（爭議立場）            [前 1-2 行]
② 反駁主流（為什麼大家錯）    [2-3 行]
③ 你的理由（1-2 點）          [2-3 行]
④ 留言鉤（請對方說服你）      [1 行]
```

### 範例

```
我覺得「斜槓」這個詞，
是這代年輕人最大的詐騙。

每個人都在學「副業思維」，
結果主業沒做好，副業也沒長出來。
真正能斜的，是先在一條路上做到 80 分的人。

不是說斜槓不好，
是太多人用它合理化「我不專精任何事」。

你怎麼看？我準備好被罵了。
```

**為什麼會爆**：
- Hook 直接挑戰當代主流敘事
- 不會太刻薄，留了「不是說斜槓不好」的台階
- 結尾「我準備好被罵了」既是邀請辯論，也是品牌人格
- 留言區會出現「+1 同意」「我不同意」兩派長文

### 應用模板

```
[爭議立場：我覺得 [反主流觀點]]

[為什麼大家錯：說出主流觀點的盲點]
[數據/例子：1-2 個支撐你的論述]

[緩衝：不是說 X 不好，是 Y]

[留言鉤：你怎麼看？/ 來說服我]
```

### ⚠️ 注意事項

- 爭議 ≠ 黑特。要有理由，不能只是情緒
- 不踩政治、種族、性別、宗教（除非品牌定位明確）
- 結尾留討論空間，不要結論式
- 預想可能的反彈，準備好你的回應（不是吵架，是補論述）

## 結構 3：故事型（漲粉勝率最高）

**邏輯**：用具體的人物 + 場景 + 轉折，讓讀者代入。**爆紅後追蹤轉換率最高**。

### 結構

```
① Hook（懸念/反差）           [前 1-2 行]
② 情境（誰、在哪、發生什麼）  [3-4 行]
③ 轉折（沒想到 / 後來）       [2-3 行]
④ 體悟（這件事讓我懂了什麼）  [1-2 行]
⑤ 留言鉤（你有類似經驗嗎？）  [1 行]
```

### 範例

```
昨天去夜市，
看到一個阿伯在賣豆花，沒有招牌。

我點一碗坐下，
他突然問我：「妳是大學生齁？」
我點頭。
「我女兒以前也是讀大學的，後來去新加坡了，
 都沒有時間回來看我。」
他停了一下，多舀了一勺豆花給我。

吃完那碗豆花，
我把行銷系開學第一堂課老師說的話想了一遍：
「行銷不是讓人買東西，
 是讓人覺得被看見。」

那一刻我才懂他在說什麼。

你曾經被一個陌生人看見過嗎？
```

**為什麼會爆**：
- 具體場景：夜市、豆花、阿伯，全部有畫面
- 情感觸動：阿伯的孤獨、想念女兒
- 觀點接續：把故事連結到品牌（行銷系）
- 留言鉤：開放、溫暖，鼓勵分享自己的故事

### 應用模板

```
[Hook：時間 + 地點 + 一個小衝突或反差]

[人物登場：誰、在做什麼]
[互動發生：對方做了/說了什麼]
[小轉折：氣氛改變的那一刻]

[體悟：這件事讓你想到什麼觀念/連結到品牌]

[留言鉤：你也有類似經驗嗎？/ 你被誰看見過？]
```

### ⚠️ 注意事項

- 故事必須真實，或起碼可信。編太誇的會被嗅出來
- 細節是靈魂：豆花、夜市、新加坡、多舀一勺——這些細節讓故事活起來
- 體悟要短，1-2 句，不要說教
- 不要在故事裡硬塞產品名稱，違和感超強

## 結構 4：清單型（收藏率最高）

**邏輯**：把資訊整理成 N 點，方便讀者收藏、轉發、抄筆記。

### 結構

```
① Hook（清單預告 + 痛點）     [前 1-2 行]
② N 點清單（每點 1-2 行）     [N × 2 行]
③ 補一句總結 / 行動建議       [1-2 行]
④ 留言鉤（你還會加哪一條？）  [1 行]
```

### 範例

```
讀行銷系 4 年，
我想告訴高中生這 5 件事：

1. 行銷不是「會做海報」，是「讓人停下來看」。
2. 文案重要，但比文案更重要的是 你寫給誰看。
3. ChatGPT 不會搶你工作，沒觀點的人才會。
4. 簡報好不好看是其次，敢不敢上台才是分水嶺。
5. 你最該學的能力叫「會說人話」。

學歷給你機會，但機會給不了你觀點。
觀點是你自己長出來的。

你覺得我漏了哪一條？
```

**為什麼會爆**：
- Hook 預告價值「我想告訴高中生這 5 件事」，TA 立刻知道為什麼要看
- 每點短而有力，易讀
- 結尾留言鉤「你覺得我漏了哪一條」邀請補充

### 應用模板

```
[Hook：N 點清單預告 + 對誰有用]

1. [短句重點 + 一句解釋]
2. [短句重點 + 一句解釋]
3. [短句重點 + 一句解釋]
4. [短句重點 + 一句解釋]
5. [短句重點 + 一句解釋]

[總結：把清單拉到更高一層]

[留言鉤：你會加哪一條？/ 哪一條最有感？]
```

### ⚠️ 注意事項

- 3-5 點最好（超過 7 點讀者讀不完會直接滑走）
- 每點不要寫太長，1-2 行為主
- 順序要有節奏：最有共鳴的放在前面或最後面
- 不要太「教學感」，要保留一點口氣

## 結構 5：反問型（互動導向）

**邏輯**：整篇貼文核心就是「問一個問題」，引導留言參與。

### 結構

```
① 鋪陳問題的背景             [2-3 行]
② 提出問題（核心）           [1-2 行]
③ 可能的選項 / 立場          [選用]
④ 你的初步立場（可選）       [選用]
```

### 範例

```
最近想了一件事：

行銷系跟資管系合起來會不會更強？

行銷系的人會「思考為什麼買」，
資管系的人會「做出讓人買的工具」。

可是一個是文組、一個是理組，
台灣的大學系所幾乎沒有真的整合的。

如果你是高中生，
你會選哪一邊？為什麼？
```

**為什麼會爆**：
- 整篇只有一個任務：請你回答
- 給了背景 + 矛盾 + 思考方向，留言成本很低
- 高中生 TA 看到會直接代入（「我會選 X 因為 Y」）

### 應用模板

```
[鋪陳：最近想了一件事 / 看到一個現象]

[問題核心：要不要 X？/ A 跟 B 哪個比較好？/ 怎麼看 Y？]

[加深問題的張力：背景補充、矛盾、可能選項]

[你的立場（可選）：我自己是覺得 X，但…]

[直接問留言]
```

### ⚠️ 注意事項

- 問題不要太大（「人生意義是什麼」沒人回）
- 也不要太小（「今天吃什麼」沒人想討論）
- 最佳：**爭議度中等、跟 TA 有切身關係**
- 不要自問自答太快，留空間給讀者

---

## 結構選擇決策樹

```
你的目標是什麼？
├── 漲粉             → 故事型
├── 留言量爆衝       → 爭議型
├── 共鳴轉發         → 共鳴型
├── 被收藏被轉發給朋友 → 清單型
└── 純互動暖場       → 反問型
```

## 結構混搭

進階玩法：

- **共鳴 + 清單**：「大學生最痛的 5 件事 ①…②…」
- **故事 + 爭議**：故事鋪陳完，丟一句反主流觀點
- **反問 + 故事**：先說故事，最後問「你會怎麼選」

---

## 寫貼文工作流（每次照做）

1. 從 Step 4 確定的「主導情緒」回推用哪個結構
2. 從 `03-hook-library.md` 找對應類型的 Hook
3. 套上面這 5 個模板的其中一個
4. 寫完跑「貼文自檢清單」（見 SKILL.md Step 5）
5. 預想 5 種留言情境（不能預想出 5 種 = 留言鉤太弱，回去改）
6. 預先寫好你要在第一則留言補充什麼
