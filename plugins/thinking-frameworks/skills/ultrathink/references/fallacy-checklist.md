# 謬誤檢核清單

整個 skill 的全程推理檢核標準——謬誤不是辯論攻防才有，任何階段寫下的推理都可能犯，所以任何框架的產出都以本清單把關：邏輯推導的推理鏈、市場分析的判斷、權衡得失的取捨理由、辯論思考的攻防、最終交付物裡的論證，全部適用。收錄約 160 條謬誤（取自維基百科[謬誤列表](https://zh.wikipedia.org/zh-tw/%E8%AC%AC%E8%AA%A4%E5%88%97%E8%A1%A8)）。對應[質詢拆解](./07-debate-thinking.md)的三個維度：多數條目屬「邏輯是否嚴密」，也有攻資料的（假資訊來源、偏差樣本等）與攻唯一性的（假兩難、單因謬誤等）。

## 檢核方式

- **檢核對象**：任何一段推理——各框架的思考筆記、對方的立論、己方準備端出去的論證或結論、任何一關的舉證。
- **輕量自檢（隨時）**：每寫完一段關鍵推理，掃一遍七組的組名與自己拿不準的條目，發現可疑就翻到該條對照。這是日常紀律，不開 subagent。
- **完整地毯式檢核（關鍵時刻）**：方向摘要收斂前、駁論、審他人的論證、重要結論端出去之前，執行完整檢核：下方七組，每組開一個獨立 context 的 subagent，各拿到待檢推理與該組的勾選清單，逐條勾選、逐條表態：確認未犯就打勾，犯了標「命中」並引述原文說明為何構成該謬誤，拿不準標「疑似」。每一條都要有結果，不跳條、不抽查，沒表態的條目視同沒檢查。組與組互不相見，避免某組的判定帶風向其他組。
- **主 agent 彙整**：去重（同一句話可能命中多個謬誤），並複核每個命中是否真的成立——謬誤指控本身也要過論點三要素：指出哪句話（資料）、犯了哪條（主張）、為什麼構成（推論）。確認的命中送回駁論或修補流程。
- **無 subagent 環境**：單人逐組依序檢核，檢完一組再進下一組，同樣逐條勾選表態，不跳條、不抽查。
- **謹記謬誤謬誤**：查出推理有謬誤，只代表該論證不成立，不代表它的結論為假。

## 第一組：形式謬誤

推理的邏輯結構有缺陷，必定無效。

### 命題邏輯謬誤

- [ ] **肯定後件**：「若P則Q」為真且Q為真，推得P為真。
- [ ] **否定前件**：「若P則Q」為真且P為假，推得Q為假。
- [ ] **肯定選言**：「P或Q」為真且P為真，推得Q為假。
- [ ] **否定聯言**：「P且Q」不為真且P不為真，推得Q為真。
- [ ] **換位不換質**：「若P則Q」為真，推得「若Q則P」為真。
- [ ] **換質不換位**：「若P則Q」為真，推得「若非P則非Q」為真。

### 量化詞邏輯謬誤

- [ ] **無端對立**：把特稱肯定句與特稱否定句互轉，從「有些S是P」推得「有些S不是P」。
- [ ] **非法換位**：從「所有S都是P」推得「所有P都是S」。
- [ ] **不當換質換位**：從「沒有S是P」推得「沒有非P是非S」。
- [ ] **存在性謬誤**：不當假定推論中提及的集合非空，從「對於所有X」推出「存在X」。
- [ ] **量化詞對調**：把「對於所有的…存在…」對調為「存在…對於所有的…」。

### 三段論邏輯謬誤

- [ ] **大詞不當**：大詞在結論中周延，在前提中卻不周延。
- [ ] **小詞不當**：小詞在結論中周延，在前提中卻不周延。
- [ ] **中詞不周延**：中詞在兩個前提中都不周延。
- [ ] **肯定前提推得否定結論**：二前提皆肯定，結論卻否定。
- [ ] **否定前提推得肯定結論**：一前提為否定，結論卻肯定。
- [ ] **互斥前提**：二前提皆為否定。
- [ ] **四詞謬誤**：因歧義等因素，三段論實際上有四個詞而非三個。
- [ ] **中詞含糊謬誤**：中詞含糊不清造成邏輯不連貫，四詞謬誤的一種。

### 雜類

- [ ] **爛理由謬誤**：基於某論證不健全，推得其結論為假。
- [ ] **謬誤謬誤**：基於某論證無效，推得其結論為假。
- [ ] **蒙面人謬誤**：對內涵性語句（相信、知道等）不當使用代換原則。

## 第二組：言詞謬誤

因語意混亂造成的不恰當推理。

### 模糊謬誤

- [ ] **劃界謬誤**：因概念沒有明確界限，就主張不應區分。
- [ ] **洛基的賭注**：主張概念無法明確定義，因而問題無法討論。
- [ ] **假精確**：把估計值當精確數值，忽略誤差。
- [ ] **具體化謬誤**：把抽象概念當具體概念使用，令人難解實際意思。

### 歧義謬誤

- [ ] **一詞多義**：字詞有多種解讀造成混淆與錯誤推理。
- [ ] **歧義句構**：句子文法上有多種理解方式造成混淆。
- [ ] **轄域謬誤**：邏輯算子作用範圍的歧義造成誤會或不當推理。
- [ ] **加重語氣**：口語抑揚頓挫不同造成的混淆與錯誤推理。
- [ ] **語境去除**：引用說詞卻忽略語境，扭曲原意。
- [ ] **斷章取義**：不當刪除上下文，扭曲原意。
- [ ] **概念滑轉**：同一字詞前後意義不同造成錯誤推理。
- [ ] **偷換概念**：蓄意把談論中的關鍵詞換成其他概念。
- [ ] **挪動門柱**：中途改變達成目標的準則。

### 定義謬誤

- [ ] **定義家謬誤**：論證時對相關詞語不當定義，含堅持使用說服性定義。
- [ ] **修改定義**：不當改變關鍵詞定義造成誤導。
- [ ] **沒有真正的蘇格蘭人**：用「真正的X」排除反例的虛假一般化。

### 廢話謬誤

- [ ] **瑣碎真理**：正確卻對主題毫無助益的資訊。
- [ ] **無意義的問題**：對認知或爭議毫無助益的發問。
- [ ] **套套邏輯（恆真句式）**：把關鍵詞換成自訂定義，使論述恆真而空洞無內容。
- [ ] **回顧性宿命論**：某事發生後，主張它是註定會發生的。

## 第三組：不一致與偽冒論題

- [ ] **自我衝突**：提出的多個主張或前提無法同時為真。
- [ ] **破釜邏輯**：防衛論點的多個理由無法同時為真。
- [ ] **矛盾的前提**：論證前提彼此矛盾，結論不可接受。
- [ ] **自我推翻**：論述者的行為或情境本身證明論述不成立，如用漢語說「我一句漢語都不會說」。
- [ ] **推論失效**：泛指前提推不出結論的情況（不相干與不充分的總稱）。
- [ ] **歪曲論題**：提出與爭議主題沒有邏輯關聯的主張或論證。
- [ ] **稻草人論證**：攻擊對方並未提出的論點。
- [ ] **紅鯡魚**：製造其他議題，轉移對原議題的注意力。
- [ ] **煙霧彈**：引入細節把議題弄複雜，令人無法注意重點。

## 第四組：偽冒理據（來源與人身）

前提和結論缺乏邏輯關聯——以來源、身分、特質、關聯當理據。

- [ ] **起源謬誤**：攻擊論證的來源，而非論證本身的有效性。
- [ ] **假資訊來源**：根據不相關、低劣、身分不明、偏頗或偽造的來源提出主張。
- [ ] **黃鼠狼的話**：訴諸身分不明的「專家」。
- [ ] **訴諸權威**：以可靠性存疑的專家背書當理據。
- [ ] **訴諸名人**：某名人支持某事，故某事是對的。
- [ ] **訴諸群眾**：很多人支持某事，故某事是對的。
- [ ] **全知論證**：宣稱「所有人都知道／都同意」。
- [ ] **訴諸人身**：以個人特質（人格、動機、地位、處境）明示或暗示其主張可取或不可取。
  - [ ] **扣帽子**：把負面特質強加至他人身上以否定其主張。
  - [ ] **人身攻擊**：宣稱某人有負面特質以否定其主張。
  - [ ] **穢言謬誤**：用不雅言語攻擊他人以否定其主張。
- [ ] **訴諸動機**：以主張背後的動機評斷主張。
- [ ] **作風蓋過實質**：以某人向來作風推斷他一定會如何。
- [ ] **你也一樣**：指對方自己也做過某事，故其反對虛偽、某事沒錯。
- [ ] **關聯謬誤**：把主張與特定形象的個人或團體連結以評斷主張。
  - [ ] **罪惡關聯**：與形象不好者連結以否定主張。
  - [ ] **希特勒歸謬法**：宣稱希特勒（或形象不好者）也支持某主張，故某主張錯。
- [ ] **訴諸好人**：以正面特質明示或暗示其主張可取。
- [ ] **訴諸成就**：以沒有相應成就否定其主張資格，如「不會開車就沒資格批評開車危險」。
- [ ] **訴諸富貴**：以有財富地位論斷主張可取，或無財富地位論斷不可取。
- [ ] **訴諸貧賤**：以貧賤為正直故主張可取，或富貴為狡詐故主張不可取。
- [ ] **訴諸智能**：以智能不足、心智不穩否定其主張。
- [ ] **詞源謬誤**：以字詞的歷史意義斷定它現在的意義。
- [ ] **格言論證**：用格言俗諺證成主張，不考慮背後理據與適用情境。
- [ ] **訴諸年代**：起於較早年代，故不如現今的好。
- [ ] **訴諸新潮**：最新、最潮，故是好的。
- [ ] **訴諸傳統**：行之已久，故是好的。
- [ ] **非我所創**：不是自己人創造的，故不好。
- [ ] **是我所創**：是自己人創造的，故好。

## 第五組：偽冒理據（情感與舉證）

- [ ] **訴諸情感**：激起他人情感以誘導支持自己的主張。
  - [ ] **訴諸恐懼**：激起恐懼誘導反對某事物。
  - [ ] **訴諸厭惡**：激起噁心厭惡誘導不支持。
  - [ ] **訴諸仇恨**：激起仇恨誘導不支持。
  - [ ] **訴諸諂媚**：奉承諂媚誘導支持自己。
  - [ ] **訴諸憐憫**：激起同情或愧疚誘導支持自己。
  - [ ] **訴諸可愛**：用可愛純真討喜的人事物誘導支持。
  - [ ] **訴諸性感**：用性感的人事物誘導支持。
  - [ ] **時髦術語**：用新奇有趣時髦的詞語誘導支持。
  - [ ] **訴諸荒謬**：嘲笑某事物荒謬可笑，誘導不支持。
- [ ] **訴諸信心**：要求對方先相信某事為真，才能找到支持它的證據。
- [ ] **訴諸自然**：合乎自然天性故好，不合故不好。
- [ ] **訴諸後果**：以命題導致的後果決定它的真假。
  - [ ] **訴諸不良後果**：因後果不欲而不相信它為真。
- [ ] **訴諸武力**：威脅懲罰，迫使對方不支持某事。
- [ ] **一廂情願**：因希望某事發生而信其為真，或不希望而信其為假。
- [ ] **訴諸無知**：未被證明為假故為真，或未被證明為真故為假。
- [ ] **訴諸難以置信**：自己難以想像某事為真，故某事為假。
- [ ] **訴諸自我知識**：自己不知道某事，故某事為假。
- [ ] **默證**：支持某事的人（或文獻）未提供證據，故某事不真。
- [ ] **證明不存在**：要求對方證明某事物不存在，否則就得承認存在。
- [ ] **轉移舉證責任**：不為己方論點舉證，要求質疑者證明它是錯的。
- [ ] **訴諸中庸**：多個衝突主張的折衷就是最佳解。
- [ ] **冤冤相報**：一個錯誤可由另一個錯誤抵消或修正。
- [ ] **我有權發表意見**：說法被質疑時，以言論自由回應而非舉證。

## 第六組：不充分的謬誤

前提與結論有關，但不足以支持結論。

- [ ] **合成謬誤**：以個體的特質推整體的特質。
- [ ] **分割謬誤**：以整體的特質推個體的特質。
- [ ] **簡化推論**：認為通則不容任何例外。
  - [ ] **偶例謬誤**：以通則否定例外的正當性。
  - [ ] **逆偶例謬誤**：以例外否定通則的正當性。
- [ ] **錯誤概括**：以不足或不具代表性的樣本歸納出一般性結論。
- [ ] **偏差樣本**：樣本太小、非隨機等缺乏代表性的樣本。
- [ ] **偏差統計**：以不恰當的統計方法推論。
- [ ] **單方論證（隱瞞證據）**：只挑支持論點的資料呈現，不支持的忽略不計。
- [ ] **軼事證據**：以傳聞、故事等少數個例推一般性結論。
- [ ] **誤導性鮮活個案**：以細節極充分的特殊個案誘導相信其普遍性。
- [ ] **壓倒性例外**：看似普遍性通則，卻附帶排除極多例外。
- [ ] **片面辯護**：主張自己是一般原則的例外，卻提出無關的理由。
- [ ] **懶於歸納**：證據強而有力仍拒絕一切歸納性結論。
- [ ] **以偏概全**：泛指以部分概括整體的推理（總稱）。
- [ ] **訴諸可能**：可能為真就當作確定為真，忽略其他可能性。
- [ ] **布佛氏論證**：先假定對方觀點是錯的，解釋為何許多人會信它，然後斷定它錯。
- [ ] **鬼祟謬誤（訴諸陰謀）**：主張真相必是有人背後操作，缺乏證據就是被掩飾。
- [ ] **不完整的比較**：用不清楚不完整的證據比較，如「更好」卻不說明標準。
- [ ] **不一致的比較**：對不同對象比較不同的事。
- [ ] **不當類比**：使用不恰當的類比推論得出不恰當的結論。
- [ ] **涅槃謬誤（完美主義謬誤）**：「不完美就不應該」，方案有缺陷因而不該做。
- [ ] **權宜主義謬誤**：「應該就別求完美」，目的重要故無論多少缺陷都該做。

## 第七組：因果謬誤與不當預設

- [ ] **偽關係**：統計上相關實為干擾因子造成，兩者無因果。
- [ ] **相關不蘊涵因果**：一起發生或正相關，就斷定甲是乙的原因。
- [ ] **後此謬誤**：乙在甲之後發生，故甲是乙的原因。
- [ ] **迴歸謬誤**：非常態事件本會回歸常態，卻歸功於中間採取的措施。
- [ ] **倒果為因**：把原因誤認為結果，或把結果誤認為原因。
- [ ] **單因謬誤**：認定某事由單一原因造成，未考慮多因共同導致。
- [ ] **無足輕重**：抓著有關但不重要的原因，遺漏更重要的原因。
- [ ] **複合結果**：把甲乙建立因果，實際上兩者都是丙造成的結果。
- [ ] **滑坡謬誤**：連串因果推論誇大每個環節的因果強度，得出不合理結論。
- [ ] **乞題（竊取論點）**：論證的前提預設了結論或其他有爭議的事。
  - [ ] **乞題類比**：類比的強度取決於原先有爭議的論點。
- [ ] **套套邏輯（迴避質疑）**：被質疑時假定原論點為真，用同義反覆的命題回應。
- [ ] **循環論證**：論點的真確性最終由自身支持。
  - [ ] **循環證成**：論點由自身支持。
  - [ ] **循環因果**：某事的原因回到某事自身解釋。
  - [ ] **循環定義**：定義某詞的詞最終需用該詞自身定義。
  - [ ] **循環解釋**：把一件事換句話說以解釋自身。
- [ ] **小小人論證**：無限迴歸式的解釋，令人無從真正理解。
- [ ] **既定觀點用詞**：描述客觀事實時，使用帶情感、評價意義的字詞。
- [ ] **假兩難推理**：主張要麼A要麼B，忽略其他可能。
- [ ] **假合取**：假定慣常被分類在一起的事物必然一起出現。
- [ ] **否認對立**：情境確實不允許其他可能時，仍主張其他可能。
- [ ] **打壓對立**：篡改對立概念的定義使其難以發生，藉以宣稱己方概念的普遍性。
- [ ] **自然主義謬誤**：把「好」定義為自然屬性、主張自然的就是好的、或以實然推應然。
- [ ] **道德主義謬誤**：「好的就是自然的」，合道德的都是自然天性。
- [ ] **證明太多**：論證還能推出更多顯然不合理的結論，依歸謬法可知論證失敗。
- [ ] **複合問題**：問題含多個子命題，卻只允許是或否的簡單回答。
  - [ ] **雙管問題**：多個子命題只許單一是否回答。
  - [ ] **既定觀點問題**：問題預設了某些事，答是答否都暗示同意。
- [ ] **誘導性提問**：用不當提問方式限縮、操弄回答。
- [ ] **訴諸斷言**：不斷聲稱自己的論點是對的，而不提供理據。
- [ ] **訴諸對立**：不斷聲稱對方論點的對立論點是錯的，而不提供理據。
- [ ] **訴諸頑固**：不斷聲稱對方論點不合理、不可信，而不提供理據。
- [ ] **訴諸反覆**：聲稱論點已充分討論過、未被駁倒，而不提供理據。
- [ ] **訴諸冗贅**：蓄意使用艱澀算式或術語，令人難以反駁。
- [ ] **亂槍打鳥論證**：提出大量論述使反對者無法一一反駁，儘管幾乎都不合理。
- [ ] **混淆充要條件**：把充分條件或必要條件當作充要條件使用。
