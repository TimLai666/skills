# 市場分析

## 主題定位

管理大師麥可・波特（Michael Porter）認為，擬定企業策略或確立市場定位之前，應該先分析內外環境，因為所謂的策略，就是「透過與其他公司相異的活動，創造具有獨特性與價值的定位」。《多模型思維》也指出，模型可以提供框架與架構，讓我們思考「如何選擇」，進而協助策畫。

套用到任何任務上：交付物不是在真空中產生的。它服務某群人、跟某些替代選項競爭注意力與選擇、也受限於自己的能力與資源。動手前先把這個環境看清楚，定位才會獨特而有價值。

## 3C 分析：因應顧客和競爭者動態，擘畫企業戰略

通用性最高的經營環境分析模型就是戰略三角，又稱 3C 分析，由日本戰略顧問大前研一提出：市場和顧客（Customer）、競爭者（Competitor）、企業自身（Company）。他認為所有戰略都要從客戶的觀點出發：必須先分析市場和顧客，再研究競爭者如何因應市場和顧客的需求變化、拉開和競爭者的差距，最後才探討自己的立場、策略及資源。一句話：**因應顧客和競爭者動態，擘畫企業戰略。**

不只商業戰略適用，任何交付物都有它的 3C：服務的對象與其需求（顧客）、爭奪同一群對象的替代選項（競爭者）、自己手上的能力與資源（自身）。寫文章時是讀者、同主題的其他內容、自己的獨特視角；做提案時是決策者、其他方案（包括維持現狀）、我方的資源強項。

三個 C 各自檢視的面向：

| C | 檢視項目 | 代表提問 |
| --- | --- | --- |
| Customer 市場和顧客 | 成長性、市場規模、區隔、顧客需求、結構變化 | 市場的規模和成長性如何？需求正往哪裡移動？ |
| Competitor 競爭者 | 進入市場的難易度、產品特徵、行銷策略 | 競爭者在哪？他們如何因應需求變化？ |
| Company 企業自身 | 技術能力、品質、銷售、獲利能力 | 自己的財務狀況和成本結構如何？強項在哪？ |

### 操作步驟

順序是這個工具的核心：從客戶觀點出發，先顧客、再競爭者、最後才是自己，不要反過來。

1. **Customer：先看市場和顧客** — 這個交付物服務誰？他們的需求是什麼、正在怎麼變化？規模與成長性如何、可以分成哪些區隔？寫下「對象要什麼」的具體判斷與依據，而不是自己想給什麼。
2. **Competitor：再看競爭者** — 誰在爭奪同一群對象？含直接對手、替代選項，以及「維持現狀、什麼都不做」。他們如何因應對象的需求變化？各自的特徵與門檻？指出對手做得好的、和做不到的地方。
3. **Company：最後看自己** — 盤點自己的立場、策略與資源：能力、品質、資源、限制。誠實列強項與弱項，對照前兩步：對象要的東西裡，哪些自己做得到、對手做不到？
4. **收斂成定位** — 找出「對象需要、競爭者沒做好、自己做得到」的交集，把定位寫成一句話：為誰、提供什麼獨特價值、憑什麼是我。三圈沒有交集就回頭調整：換區隔、補能力，或重新選戰場。

### 常見錯誤

- 順序反了：先從自己有什麼出發，再找理由說服對象買單。3C 的鐵律是從顧客觀點出發，自己放最後。
- 顧客分析寫成自己的想像：「他們應該想要」卻沒有依據。要看需求、區隔與變化的實際訊號。
- 競爭者只算同類對手：漏掉替代選項和「維持現狀」。搶走對象的，不一定長得像你。
- 只分析不收斂：三個 C 各寫一堆，沒有推出定位。分析的目的是找到與眾不同、獨特而有價值的位置。
- 定位選在自己強但對象不要的地方：要交集三個圈，不是只放大自己的強項。
- 一次分析用到老：顧客需求和競爭者都在動態變化，定位要跟著環境更新。

### 自檢清單

做完 3C 分析後逐項檢查，任何一項不成立就回去補：

- [ ] 分析順序是顧客 → 競爭者 → 自己，沒有從自己出發
- [ ] 對象的需求、區隔與變化有具體判斷和依據，不是想像
- [ ] 競爭者盤點含直接對手、替代選項與「維持現狀」
- [ ] 有指出競爭者做得好與做不到的地方
- [ ] 自身盤點誠實列出強項、弱項、資源與限制
- [ ] 有收斂出一句話定位：為誰、提供什麼獨特價值、憑什麼是我
- [ ] 定位落在「對象要、對手弱、自己強」的交集上

## SWOT 分析：審視內外部變因，辨識機會和威脅

SWOT 分析從內、外部環境分析現狀，代表優勢（Strength）、劣勢（Weakness）、機會（Opportunity）及威脅（Threat）。優劣勢是跟競爭者比較的結果；機會與威脅則是所處的環境。分析時從外部環境（機會與威脅）開始，分為宏觀及微觀兩層，再分析內部環境（優勢與劣勢）。活用優勢贏得機會，就能讓成果成長。一句話：**審視內外部變因，辨識機會和威脅。**

任何要評估「現在站在什麼位置、該往哪走」的時刻都適用：組織、專案、方案、產品，甚至自己的處境。承接 3C 很自然：3C 看完顧客、對手與自己之後，SWOT 把觀察整理成內外×正負的四格，導出行動方向。

若環境裝有本系列 business-strategy plugin 的 `swot-analysis`（完整策略工作流：環境掃描、S/W/O/T 驗證、SO/ST/WO/WT 策略碰撞）與 `pestel-analysis`（宏觀環境系統掃描），建議一併載入搭配，把這裡的產出接進更完整的策略流程；沒有的話，本節流程即可獨立完成。

| | 正面因素 | 負面因素 |
| --- | --- | --- |
| **內部環境**（自己可控、與競爭者相比） | 優勢 Strength | 劣勢 Weakness |
| **外部環境**（環境所給、不因你而存在） | 機會 Opportunity | 威脅 Threat |

### 操作步驟

1. **定分析主體與比較基準** — 這次分析的是什麼（組織、方案、產品、自己）？優劣勢要跟誰比（競爭者、替代選項）？沒有比較基準，優劣勢會寫成自我感覺。
2. **先掃外部：機會與威脅** — 列出所處環境的正面與負面因素，分兩層掃：宏觀（大環境趨勢：政策、經濟、社會、技術等，可搭配 PESTEL 系統化展開）與微觀（身邊的環境：對象、對手、上下游）。外部因素的判準：不管你存不存在，它都在發生。
3. **再看內部：優勢與劣勢** — 跟比較基準相比，自己強在哪、弱在哪。內部因素的判準：是自己可控、可改變的。每一條都寫得出「比誰、好或差在哪」。
4. **填四格並逐條歸位檢查** — 每條因素問兩題：這是內部還外部（可控嗎）？正面還負面？同一條不能既是機會又是優勢——「手機下單的人增加」是環境給的（機會），「網站介面比對手順暢」是自己做到的（優勢）。
5. **收斂成行動方向** — 至少回答：如何活用優勢贏得機會（S×O）？劣勢撞上威脅時怎麼補救或避開（W×T）？把四格交叉成方向，不要填完就結束。要做完整的 SO/ST/WO/WT 策略碰撞，搭配 `swot-analysis` skill。

### 對照範例：鞋類品牌電商

| 象限 | 內容 |
| --- | --- |
| 優勢 | 網站介面比競爭對手順暢、與熱門 IP 合作開發聯名商品、擁有實體店面 |
| 劣勢 | 忠誠會員較少、商品數量較少 |
| 機會 | 透過手機下單的人增加、藝人公開說愛用自家品牌、顧客單價提高 |
| 威脅 | 品牌 A 以低價走紅、無法做到精準個人化行銷、流行鞋款多庫存壓力大 |

兩個觀察：優勢第一條「比競爭對手順暢」自帶比較基準，是優劣勢的標準寫法。而威脅欄的「無法做到精準個人化行銷」用步驟 4 檢查會卡住——「無法做到」是自身能力缺口，歸劣勢更準；要當威脅，應寫成「精準個人化行銷成為市場標準」。範例照抄也要驗，這正是歸位檢查的用途。

### 常見錯誤

- 優劣勢沒有比較基準：「我們品質好」——比誰好？優劣勢是跟競爭者比較的結果。
- 內外不分：把自己的能力缺口寫成威脅、把環境紅利寫成優勢。判準是可控性。
- 順序反了：先寫自己再找環境背書。先看清環境，再評自己在其中的位置。
- 機會寫成願望清單：機會是環境給的，不是「自己想做的事」。
- 每格塞好塞滿：不分輕重，重要的少數被雜訊淹沒。
- 只列不收斂：四格填完就交卷，沒有推出行動方向，活用優勢贏得機會才是目的。

### 自檢清單

做完 SWOT 分析後逐項檢查，任何一項不成立就回去補：

- [ ] 分析主體與比較基準明確，每條優劣勢都寫得出「比誰」
- [ ] 順序是先外部（機會、威脅，宏觀＋微觀兩層）再內部（優勢、劣勢）
- [ ] 每條因素歸位檢查過：可控的在內部，環境給的在外部
- [ ] 沒有一條同時當機會又當優勢，或同時當威脅又當劣勢
- [ ] 每格留下的是重要的少數，不是塞好塞滿
- [ ] 至少回答了「如何活用優勢贏得機會」
- [ ] 有收斂成行動方向，不是填完四格就結束
