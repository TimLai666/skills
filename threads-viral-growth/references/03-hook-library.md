# 03｜Hook 庫｜50+ 種開頭模板

> Hook = 貼文的第一行。決定 70% 的生死。

## Hook 的功能

Hook 不是「漂亮的開場」，是「停止滑動的拳頭」。讀者在 0.5 秒內判斷要不要繼續看，**這 0.5 秒就是 Hook 的戰場**。

## Hook 三條鐵律

1. **25 字以內**（理想 12-18 字）。太長一定被滑過。
2. **必須有「鉤點」**：疑問、衝突、痛點、反差、好奇心、立場挑釁，至少佔一個。
3. **拒絕「概念詞」開頭**：「分享」「教學」「整理」「重點」「乾貨」這種詞放在第一行 = 自殺。

## Hook 五大類型 + 範例庫

### 🅰️ 疑問型 Hook（引發好奇）

用問句製造資訊缺口，讀者想知道答案就會繼續看。

**句型公式**：
- 真的有人 [行為] 嗎？
- 你有沒有發現 [現象]？
- 為什麼 [大家都這樣]？
- [某事] 是不是只有我會這樣？
- 有人懂 [小眾感受] 嗎？

**範例**：
1. 真的有人每天不喝飲料就活不下去嗎？
2. 你有沒有發現，最近大家都在說自己很累？
3. 為什麼 25 歲開始就覺得時間不夠用？
4. 一個人吃飯到底是治癒還是孤獨？
5. 有人也是看到 IG 限動會焦慮的嗎？
6. 為什麼台灣的早餐店要叫「美而美」？
7. 真的有人會把指甲剪到手痛還繼續剪嗎？
8. 是不是只有我覺得，加班加到沒人懂我笑話的時候最孤單？
9. 為什麼大學生越來越愛去學校附近的便利商店？
10. 有沒有一種人，他傳訊息只回「嗯」就讓你想分手？

**用法重點**：
- 問句不要太大（「人生是什麼？」太空）
- 也不要太小（「你吃飽了嗎？」沒有共鳴）
- 最佳：**具體 + 共感** 的問題

---

### 🅱️ 吐槽型 Hook（共感發洩）

用幽默諷刺切入，讓讀者「就是這樣！」想留言。

**句型公式**：
- 最累的不是 [X]，是 [Y]
- [X] 根本是騙人的
- 我真的受夠 [社會普遍現象]
- 沒有人會告訴你 [真相]
- [行業/角色] 不會說的秘密

**範例**：
11. 最累的不是報告，是組員說「我都可以」
12. 比學測更可怕的，是不知道自己適合什麼
13. 健身房年費 12,000，去三次的人比較虧還是去 300 次的比較傻？
14. 沒有人會告訴你，畢業後你最常用的是 Excel 不是 PowerPoint
15. 「我來幫你看一下」這句話，是辦公室最大的謊言
16. 老闆說「下班就下班」的時候，你就知道今天要加班了
17. 「我們再 confirm 一下」的意思是「你再做一遍」
18. 最痛的不是分手，是發現他換了大頭貼還比你好看
19. 跑步機上跑 30 分鐘，比通勤回家還累
20. 滑了 2 小時 Threads 之後，比念書還累

**用法重點**：
- 吐槽要有畫面感（看到立刻浮現場景）
- 不要太憤怒，要「白眼級」幽默
- 配合品牌人格：太刻薄會傷品牌，太溫和又沒共鳴

---

### 🅲 共鳴型 Hook（說出心聲）

寫出 TA 心裡早就有、但沒有人幫他講出來的那句話。**這是爆文勝率最高的類型**。

**句型公式**：
- 很多人以為 [X]，但其實 [Y]
- 我們都經歷過 [情境]
- 沒有人說，但大家都在 [行為]
- [某個年齡/身份] 才懂的事
- 偷偷說，我也 [秘密]

**範例**：
21. 很多人以為行銷系只是在做海報
22. 大學最累的不是課業，是不知道自己未來要幹嘛
23. 沒有人說，但大家都在偷偷比較同學的實習公司
24. 23 歲才懂的事：你媽說的「我沒事」永遠不是真的沒事
25. 偷偷說，我滑 Threads 滑到三點是因為失眠
26. 我們都在裝自己沒事，然後在凌晨三點傳訊息給不該傳的人
27. 出社會才懂：學校教的不是知識，是「被罵了還能繼續做」的能力
28. 我以為長大會有答案，結果只有更多問題
29. 沒有人告訴你，談戀愛最累的不是吵架，是冷戰
30. 大學生的存款，是用便當數量計算的

**用法重點**：
- 必須真實。寫得出來的人，自己一定經歷過
- 越具體越好（「23 歲」比「年輕時」共鳴更強）
- 不要寫「人生」「夢想」這種大詞，越具體的小情境越打人

---

### 🅳 爭議型 Hook（刺激立場）

提出反主流觀點，激發認同/反對兩派來辯論。**留言量最高的類型**，但要小心傷品牌。

**句型公式**：
- [大眾觀點] 是錯的，我說一個你不愛聽的
- 拒絕 [主流] 才是 [正確的]
- 老實說，[爭議觀點]
- 大家都在說 X，但真的是這樣嗎？
- 我知道講這個會被罵，但 [立場]

**範例**：
31. 老實說，免費的東西通常都最貴
32. 我覺得 25 歲還跟父母拿錢沒什麼好丟臉的
33. 「努力就會成功」這句話，騙了我整個 20 歲
34. 大家在罵慣老闆的時候，先想想你是不是慣員工
35. 不接受加班的人，不是懶，是清醒
36. 我覺得創業是這代年輕人最大的詐騙
37. 把自己練得很厲害不會讓你變快樂，會讓你更孤單
38. 「斜槓」這個詞被用來合理化「沒有專長」
39. 你的人脈不會幫你，會幫你的是你的作品
40. 不去聽演講不會錯過什麼，去聽了也通常沒記得什麼

**用法重點**：
- 爭議要有道理，不能只為了爭議而爭議
- 不要踩政治、種族、性別這種敏感線（除非品牌定位明確就是這個角度）
- 留好台階：可以強硬，但結尾要留討論空間（不要結論式）

---

### 🅴 反差型 Hook（打破預期）

用前後落差製造驚訝，讓讀者「等等什麼？」想看完。

**句型公式**：
- 我以為 [X]，直到 [Y]
- 沒想到 [意外結果]
- [預期] vs [現實]
- 表面上 [A]，實際上 [B]
- 我花了 [時間] 才發現 [真相]

**範例**：
41. 我以為去 7-11 是因為近，後來才發現是因為店員不會問我為什麼
42. 我以為大學會交到一輩子的朋友，畢業後才發現連 IG 都不互追
43. 沒想到讓我撐過大學的不是教授，是樓下的鹽酥雞
44. 我花了 4 年才發現，行銷系最重要的不是創意，是會講故事
45. 表面上在學「品牌策略」，實際上在學「怎麼讓人停下來看你」
46. 以為長大是變得堅強，後來才發現是變得會藏
47. 我以為焦慮會隨時間消失，27 歲才發現它只是換了個名字
48. 我以為 ChatGPT 會搶走我的工作，沒想到搶走的是我的耐心
49. 學校教的是「怎麼回答問題」，社會問的是「你會問什麼問題」
50. 我以為失敗是一件事，後來發現是一種狀態

**用法重點**：
- 反差要真實，不要為了反差而反差
- 前半句要設「合理預期」，後半句才能驚到人
- 配合品牌：什麼領域的反差最能彰顯品牌專業

---

## 進階：複合型 Hook

把兩種類型疊在一起，威力更強。

- **共鳴 + 反差**：「我以為大學生最焦慮的是課業，後來才發現是組員不讀訊息」
- **疑問 + 吐槽**：「為什麼老闆每次說『簡單講一下』後面都接 30 分鐘？」
- **爭議 + 反差**：「我覺得不努力的人有時候比努力的人更聰明，因為他們知道哪些事不值得做」

---

## Hook 自檢清單

寫完 Hook，跑一次：

```
□ 字數 ≤ 25 字
□ 第一行就有「鉤點」（疑問/衝突/痛點/反差/挑釁）
□ 沒有用「分享」「教學」「整理」「重點」這種廢詞開頭
□ 一個陌生人滑到，會停下來
□ 配得上後面要講的內容（不能標題黨）
```

## 如何根據趨勢量產 Hook

工作流：

1. 看當下趨勢話題的主要情緒（憤怒？好奇？共鳴？）
2. 對應到上面五大類型
3. 把品牌/任務的關鍵字代入該類型的句型公式
4. 一次寫 5-10 個 Hook，選最好的 1-2 個
5. 自檢清單跑過

**例子**（如果趨勢是「大學申請放榜」）：

```
情緒：焦慮、共鳴、不確定
類型選擇：共鳴型 + 反差型

候選 Hook：
1. 真的有人 18 歲就知道自己要做什麼嗎？（疑問型）
2. 比放榜更可怕的，是放榜後不知道下一步（吐槽型）
3. 沒有人告訴你，選系不會決定你的人生，但會決定你大學四年怎麼活（共鳴型）
4. 我以為選錯系是世界末日，後來才發現轉系比想像中容易（反差型）

選定：第 3 個（最有觀點、可討論度高）
```

## 救援：Hook 寫不出來時

如果卡關：

1. 倒過來寫：先寫完整貼文，最後一句往往是最強的 Hook
2. 問自己：「如果我只能講一句話，我會講什麼？」
3. 看留言區：競品爆文的留言常常藏著更好的 Hook 句
4. 把 Hook 講給朋友聽，看他的反應
5. 用 Claude / GPT：丟主題進去，請它用上述五種類型各生 3 個 Hook，再挑
