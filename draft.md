# 商業企劃寫作skill 草稿

## 核心認知
企劃書是讓決策者說Yes的**商業說服文件**。

## 商業企劃寫作類型
三種情境，三種寫法
1. 內部提案 -> 說服主管投入資源
2. 外部募資 -> 說服投資人掏錢
3. 合作提案 -> 說服夥伴一起做

## 企劃書結構
1. 執行摘要
2. 問題定義
3. 目標客群
4. 競爭分析
5. 方案模式
6. 行銷策略
7. 執行團隊
8. 財務風險

## 動筆前的策略思考引擎
動筆前一定要先思考。先花70%時間想，30%時間寫。
### 企劃五問
1. 為誰解決什麼問題？ -> 精準定義痛點與對象
2. 為什麼是現在？ -> 建立時機的急迫性
3. 憑什麼是我們？ -> 建立團隊的可信度
4. 要花多少、賺多少？ -> 建立投資的合理性
5. 成功長什麼樣？ -> 建立願景的吸引力

## 寫作技巧
### 執行策略摘要
「主管只看這一頁——你怎麼讓他看完就想說Yes？」

執行摘要三層結構：
1. **痛點開場** -> 建立急迫感
    e.g.,「目前客戶流失率23%，每月損失$180K⋯」
2. **方案亮點** -> 建立信心
    e.g.,「透過AI 預測模型，可降低60% 流失風險⋯」
3. 數字收尾 -> 建立行動力
    e.g.,「預計一年內回收投資，三年內創造$5M 營收⋯」

**地雷**：避免用「本企劃旨在⋯」「為了提升⋯」這種空洞開場。用數據、對比、結果開頭。

### 問題定義與市場論述
「讓人覺得『這件事非做不可』的寫法」

市場論述技巧：不是貼產業報告，而是 「用3個數據點說服人」— **市場規模** + **成長率** + **未被滿足的需求缺口**

問題定義四元素：
1. **誰的痛點** -> 精準定義受影響的對象，不是「大家」，是「誰」
2. **痛多大（量化）** -> 用數字說明問題嚴重程度，金額、比例、時間損失
3. **現有方案為何不行** -> 現在人們怎麼解決？哪裡做得不夠好？
4. **不做會怎樣** -> 建立「不行動」的成本，讓決策者感到急迫

### 目標設定與客群拆解
「不是列KPI，而是讓人相信你的KPI」

目標反推法：
1. 設定終局目標（例：年營收3000萬）
2. 拆解需要幾個客戶×客單價
3. 反推每月需要多少商機
4. 推算需要多少曝光與轉換
5. 每個階段的條件與資源需求

客戶決策旅程地圖：
1. 誰買單？（決策者）
2. 誰影響決策？（影響者）
3. 誰使用？（使用者）
4. 他們在意什麼？（決策因素）
5. 他們的替代方案是什麼？

### 競爭分析與差異化定位
「不是填SWOT，而是找到『為什麼選我』的答案」

寫作技巧：不是列出所有競品，而是選擇2-3 個最有參考價值的對比對象，塑造「我們的位置」

競爭分析三步驟：
1. **現況掃描** -> 客戶現在怎麼解決這個問題？列出所有替代方案（包含「不做」）
2. **痛點定位** -> 現有方案哪裡不夠好？找出具體的不滿與缺口
3. **優勢鎖定** -> 我們在哪個關鍵維度上能贏？塑造「競爭矩陣」中的位置

產出：**競爭矩陣**、**一句話差異化宣言**

### 方案設計與商業模式
「你賣的到底是什麼？客戶買的到底是什麼？」

價值階梯寫法—客戶不買「功能」，客戶買「成果」：
- **功能層** -> 我們提供什麼 e.g., AI數據分析平台
- **好處層** -> 對客戶有什麼用 e.g., 即時掌握市場動態，節省80%分析時間
- **成果層** -> 客戶最終得到什麼 e.g., 年度決策準確率提升40%，多創造$5M營收

商業模式三問：
客戶為什麼付錢？×付多少？×持續付多久？

### 行銷策略與推廣計畫
「不是列通路，而是說服人『這個推廣計畫會有效』」

預算寫法：用「單位經濟效益」而非總數—算CAC / LTV / ROAS，讓主管看到每一塊錢的回報邏輯

行銷漏斗寫作法：
1. **曝光** -> 客戶從哪裡來？ 指標：觸及數、曝光量
2. **接觸** -> 怎麼建立第一次互動？ 指標：點擊率、註冊率
3. **轉化** -> 怎麼把興趣變成訂單？ 指標：CVR、CAC
4. **留存** -> 怎麼讓客戶持續付費？ 指標：留存率、LTV

### 執行計畫與團隊論述
「不是列甘特圖，而是讓人相信『你們做得到』」

團隊論述重點：不是列組織圖，而是說明「為什麼這個團隊能成功」—過往成績×關鍵能力×資源優勢

里程碑寫法—標示「關鍵決策點」與「階段性成果」：
1. 準備期（第1-2個月） -> 完成MVP + 種子用戶測試 Go/No-Go 決策點
2. 驗證期（第3-4個月） -> 首批50付費客戶 規模化投資決策
3. 成長期（第5-8個月） -> 月營收突破百萬 團隊擴編決策
4. 擴張期（第9-12個月） -> 建立品牌護城河 下一輪策略規劃

### 財務預估與風險評估
「讓數字說話，但要說得讓人相信」

假設透明化寫法：
- 不夠的寫法：「預估第一年營收500萬」
- 可信的寫法：「基於X個客戶×客單價Y×年購買頻次Z = 500萬」

主動揭露寫法：
1. 選3個最可能發生的風險
2. 寫出「如果發生，我們會怎麼做」
3. 這反而建立信任—表示你想過了

產出：
1. 假設透明化財務表—每個數字都有推導依據
2. 前三大風險因應計畫—展現「我們想過最壞的情況」的專業度

## 統整與簡報演練
「寫得出來也要講得出來——讓企劃書『活起來』」

### 企劃書健檢表
- 邏輯鏈是否完整（Why → What → How → Proof → Ask）
- 數據是否前後一致（財務、目標、行銷數字互相吻合）
- 語言是否專業但不生硬（讀起來流暢、不是翻譯腔）
- 視覺是否清晰（圖表輔助理解，而非裝飾）

### 10分鐘提案結構
- **開場鉤子**(2min)：用一個數據或故事抓住注意力
- **問題與方案**(3min)：痛點->解法->差異化
- **數字與證據**(2min)：財務模型+里程碑
- **要求與下一步**(3min)：明確說出你需要什麼
