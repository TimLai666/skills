# CBBE 金字塔與 Aaker 識別稽核

## 使用規則

- CBBE 診斷「由下而上」逐層檢查：salience → performance & imagery → judgments & feelings → resonance。
- 卡關層 = 由下往上「第一個不健康的層」。下層沒過，上層數據再好看也可能是倖存者偏差（會回答問卷的都是死忠戶）。
- 每層先看有沒有數據；沒有就標「需補測」並附量測方法，不得用印象代替。
- 最終結論只指向「單一主要卡關層」。資源永遠有限，四層都修等於都不修。

## Layer 1：Salience 顯著性

問題：「顧客在需要的時候，想得起你嗎？」

量測指標：

| 指標 | 量測方法 | 判讀 |
|---|---|---|
| unaided awareness | 「講到 [品類]，你想到哪些品牌？」開放作答，記提及率與提及順序（top of mind = 第一提及） | 與前三競品比排名 |
| aided awareness | 「你聽過 [品牌] 嗎？」勾選式 | aided 高、unaided 低 = 認得但想不起，廣度不足 |
| category cue 聯想測試 | 給情境線索不給品類名：「加班到十點想吃點什麼的時候，你會想到？」看品牌是否進入該場景的候選集 | 測 breadth：品牌在多少個使用情境裡被想起 |

- depth（想得起來的強度）與 breadth（在多少情境被想起）分開判讀。成熟品牌的成長空間常在 breadth：認知度已高，但只綁定單一場景。
- unaided / aided 比值過低（認得的人很多、想得起的人很少），介入方向是「品類線索連結」而非「曝光量」。

## Layer 2：Performance & Imagery 效能與形象

問題：「顧客覺得你是什麼、表現如何？」

### Performance 量測

用 1-7 分屬性評分，「同時測 2-3 個競品」——絕對分數無意義，相對位置才有。五個構面與題目示例：

| 構面 | 題目示例 |
|---|---|
| 主要功能屬性 | 「[品牌] 在 [品類核心功能] 上的表現？」（品類不同，核心屬性清單不同，先做屬性引出訪談再定題） |
| 可靠性 / 耐用性 / 可維修性 | 「它每次表現一致嗎？」「用多久會壞？」「出問題好處理嗎？」 |
| 服務效果 / 效率 / 同理心 | 「問題有被解決嗎？」「等多久？」「對方在乎你的狀況嗎？」 |
| 風格與設計 | 「外觀 / 介面給你什麼感覺？」 |
| 價格與價值感 | 「以這個價格，值得嗎？」（測 value for money，不是便宜與否） |

### Imagery 量測

- 使用者形象聯想：「你覺得用這個品牌的人是什麼樣的人？」自由描述後編碼。品牌想吸引的人 vs 顧客描述的人，落差即問題。
- 投射法：「如果這個品牌是一個人，他幾歲、做什麼工作、週末在幹嘛？」繞過理性防衛，抓出真實人格聯想。
- 購買與使用情境聯想：「什麼時候會用它？」對照品牌想佔據的場景。
- 歷史與傳承聯想：品牌故事有沒有被記住、記對。

## Layer 3：Judgments & Feelings 評價與感受

問題：「顧客怎麼評價你、對你有什麼感覺？」

### Judgments 四項

| 項目 | 量測問題 |
|---|---|
| quality 品質 | 整體品質評分、與價格對照的價值評分 |
| credibility 信譽 | 三子項分測：專業（很懂這行嗎）、可信（說到做到嗎）、討喜（喜歡這家公司嗎） |
| consideration 考慮 | 「下次購買 [品類] 時，會把它放進候選嗎？」——比好感度更硬的指標 |
| superiority 優越 | 「它有沒有其他品牌給不了的東西？」開放作答，答不出來 = 無 POD 認知 |

### Feelings 六種品牌感受

| 感受 | 類型 | 內涵 | 典型品類 |
|---|---|---|---|
| warmth 溫暖 | 即時體驗型 | 平靜、窩心、感性 | 家庭食品、賀卡、在地老店 |
| fun 樂趣 | 即時體驗型 | 好玩、輕鬆、愉快 | 零食、遊戲、娛樂 |
| excitement 興奮 | 即時體驗型 | 有活力、酷、被激發 | 運動、潮流、新科技 |
| security 安心 | 持久私密型 | 安全、有保障、可依靠 | 保險、金融、醫療 |
| social approval 社會認同 | 持久私密型 | 用它讓別人對我另眼相看 | 精品、名車、禮品 |
| self-respect 自我尊重 | 持久私密型 | 用它讓我覺得自己更好 | 健身、學習、永續品牌 |

量測：受訪者從情緒詞列表勾選 + 強度評分。判讀規則：後三種與忠誠的相關高於前三種；只有 fun/excitement 沒有 security/self-respect 的品牌，容易被下一個新鮮品牌取代。

## Layer 4：Resonance 共鳴

問題：「顧客願不願意跟你維持關係、為你發聲？」

| 構面 | 指標 |
|---|---|
| behavioral loyalty | 回購率、購買頻次、share of wallet（品類支出中你的占比） |
| attitudinal attachment | NPS、「如果它消失了你會在意嗎」 |
| sense of community | 社群成員數與活躍率、成員間互動（不只品牌對粉絲，還有粉絲對粉絲） |
| active engagement | UGC 量、主動推薦次數、非購買期的互動（追蹤、討論、參加活動） |

判讀：behavioral loyalty 高但 attitudinal attachment 低 = 假共鳴（可能只是通路方便或轉換成本高），競品一補貼就流失。

## 卡關症狀 → 介入手段對照表

| 卡關層 | 症狀 | 常見根因 | 介入手段 |
|---|---|---|---|
| salience | 沒人想得起；aided 高 unaided 低 | 曝光不足，或曝光了但沒綁品類線索 | 觸及投資 + 品類情境連結；建立 distinctive assets（顏色、聲音、吉祥物） |
| performance | 想得起但屬性評分輸競品 | 產品力真的不夠 | 先改產品再溝通。溝通無法修產品問題 |
| imagery | 功能評分 OK 但形象模糊或不符目標 | 溝通只講功能沒建人設；使用者形象被非目標客群定義 | 使用者形象溝通、聯名、場景行銷、代言選擇 |
| judgments | 知道也覺得不錯，但不信任、不進候選集 | credibility 缺口：承諾與體驗不一致、負評未處理 | 第三方背書、透明化、保證機制、負評回應 SOP |
| feelings | 理性認可但無感情，「還不錯，但沒感覺」 | 體驗無記憶點、無故事 | 體驗設計、品牌敘事、儀式感設計 |
| resonance | 買過不回頭、無人推薦 | 購後被遺忘、無社群機制、回購動線斷裂 | 忠誠計畫、社群經營、回購提醒設計、UGC 激勵 |

判讀警告：

- 高知名度 + 低評價 ≠ 溝通問題，通常是產品或體驗問題。加碼廣告只會加速讓更多人知道你不好。
- 低知名度 + 高評價（在既有顧客中）= 最值得加碼觸及的訊號。

## 診斷問卷題目範例

依層抽題組卷，每層至少 2 題，全問卷同時測 2-3 競品：

1. 講到 [品類]，你第一個想到哪個品牌？還有呢？（salience-unaided）
2. 以下品牌你聽過哪些？（salience-aided）
3. [情境描述] 的時候，你會想到哪些選擇？（salience-breadth）
4. 請為 [品牌] 的 [核心屬性] 打 1-7 分。（performance）
5. 你覺得用 [品牌] 的人是什麼樣的人？（imagery-使用者形象）
6. 如果 [品牌] 是一個人，請描述他。（imagery-投射）
7. 你覺得 [品牌] 說到做到嗎？為什麼？（judgments-credibility)
8. 下次買 [品類] 時，[品牌] 會在你的考慮名單裡嗎？（judgments-consideration）
9. [品牌] 有沒有什麼是其他品牌給不了的？（judgments-superiority）
10. 用 [品牌] 的時候，你有什麼感覺？請從清單勾選。（feelings）
11. 過去一年你買 [品類] 幾次？其中 [品牌] 幾次？（resonance-SOW）
12. 你會推薦 [品牌] 給朋友嗎？0-10 分。（resonance-NPS）
13. 如果 [品牌] 明天消失，你的感覺是？（resonance-attachment）
14. 你有沒有主動分享、評論或推薦過 [品牌]？（resonance-engagement）

## Aaker 品牌識別系統

CBBE 測的是「顧客腦中實際的 image」；Aaker 定義的是「品牌想成為的 identity」。稽核 = 比對兩者落差。

### Core vs Extended Identity

- **core identity**：品牌的靈魂，跨市場、跨產品線、跨十年不變。2-4 條。判斷法：「如果放棄這一條，還是同一個品牌嗎？」答「不是」的才進 core。
- **extended identity**：補足 core 的細節（人格、slogan、視覺、產品線主張），可隨市場調整。core 塞太多條 = 什麼都核心等於沒有核心。

### 四視角盤點表

| 視角 | 盤點問題 |
|---|---|
| brand as product | 綁定哪個品類？主張哪些屬性與品質？佔據哪些用途場景？誰在用？產地或出身重要嗎？ |
| brand as organization | 組織屬性是什麼（創新、在地、永續、職人）？這些屬性顧客感受得到嗎？在地還是全球定位？ |
| brand as person | 品牌人格是什麼（接 03 參考文件的原型）？跟顧客是什麼關係——朋友、專家、教練、僕人？ |
| brand as symbol | 有哪些視覺資產（logo、色彩、包裝、吉祥物）？有沒有可用的隱喻？有沒有值得說的傳承？ |

規則：不必四視角全填，但至少 product + 另一視角。只有 product 視角的品牌 = 功能供應商，最容易被比價。

## 品牌資產稽核執行步驟

### 1. 內部訪談

高層題組：

- 用一句話說，這個品牌的核心是什麼？（每位高層分開問，答案分歧度本身就是診斷數據）
- 五年後你希望顧客怎麼描述我們？
- 哪些東西絕對不能變？哪些可以變？（core vs extended 的原始素材）

一線員工題組：

- 顧客最常稱讚什麼？最常抱怨什麼？
- 你私下怎麼跟朋友介紹這家公司？
- 公司說的品牌主張，你在日常工作裡做得到嗎？哪裡做不到？

要抓的落差：高層說的 identity vs 一線實際傳遞的訊息。高層答案彼此分歧 = identity 未建立，稽核結論須先處理這件事。

### 2. 顧客調查

用上方問卷量化 + 8-12 位顧客深訪（含流失顧客 2-3 位——流失戶的答案比忠誠戶誠實）。

### 3. 競品對照

同一組指標測 2-3 競品。聯想地圖作法：把各品牌的前十大自由聯想並列，標出「獨有聯想」與「共有聯想」。與競品重疊度高 = 無差異化認知，回 01 參考文件重跑 POD。

### 4. 落差分析

identity（想要的）vs image（實際的），落差分三類，解法完全不同，必須標明是哪一類：

| 落差類型 | 訊號 | 解法方向 |
|---|---|---|
| 溝通落差 | 體驗好的顧客描述得出 identity，沒體驗過的完全無感 | 溝通投資與訊息聚焦 |
| 體驗落差 | 顧客聽過主張但實際體驗對不上，「說一套做一套」 | 先修營運與體驗，暫停放大溝通 |
| 認知時滯 | 品牌已改變，顧客印象停在舊版本 | 需要時間 + 標誌性事件（signature move）加速改觀 |

### 5. 輸出

卡關層 + 落差類型 + 介入優先序（附每項介入的量測指標與回測時點）。scorecard 格式：

```
| CBBE 層 | 狀態(健康/偏弱/卡關) | 證據或「需補測」 | 相對競品位置 |
主要卡關層：（單一）
落差類型：（溝通/體驗/認知時滯）
介入計畫：（優先序 | 介入項 | 對應指標 | 回測時點）
```

## 本檔案判斷的失效條件

- 「由下而上找第一個不健康層」在雙客群品牌（如平台的買賣兩端）可能失效——兩端要分開跑金字塔，不可混測。
- 問卷樣本若只來自現有顧客名單，salience 與 judgments 會系統性高估，結論只適用留存客不適用市場。
- 新品牌（上市一年內）不適用完整稽核，resonance 層必然空白，只跑下兩層即可，不要把「還沒長出來」判成「卡關」。
